## 一、主角设定

### 1.1 基本信息

| 属性 | 设定 |
|------|------|
| **姓名** | 玩家自定义（2-6 字） |
| **现代姓名** | 项目经理（默认姓林，名可自定义） |
| **年龄** | 28 岁（现代）→ 18 岁（穿越后） |
| **职业** | 项目经理（996 社畜，加班猝死） |
| **穿越方式** | 随机（每次开局不同） |
| **初始属性** | 随机（根据穿越方式有差异） |

### 1.2 程序员专属梗

| 场景 | 台词/内心 OS |
|------|-------------|
| 第一次看到武功秘籍 | "这玩意比代码还难懂..." |
| 修炼内力 | "原来这就是人体的 API 接口" |
| 遇到 Bug 级敌人 | "这 Boss 有外挂吧！" |
| 队友问他在想什么 | "我在想这个武功能不能封装成函数" |
| 最终决战前 | "来吧，这就是我的 Final Battle" |
| 表白场景 | "你是我生命中最美的 Bug" |

### 1.3 初始属性随机范围

**穿越方式随机池：**

| 穿越方式 | 概率 | 属性加成 |
|----------|------|----------|
| **加班猝死** | 30% | 悟性 +10，根骨 -5 |
| **意外触电** | 20% | 内力 +15，身法 -5 |
| **坠崖** | 20% | 根骨 +10，外貌 -5 |
| **神秘古物** | 15% | 悟性 +15，随机负面 |
| **车祸** | 10% | 全属性 +5 |
| **雷劈** | 5% | 全属性 +10（欧皇） |

**属性随机范围：**

| 属性 | 最小值 | 最大值 | 程序员基准 |
|------|--------|--------|------------|
| 根骨 | 5 | 15 | 8（久坐腰不好） |
| 内力 | 5 | 15 | 10（熬夜肾还行） |

#### 🌸 苏婉儿（冷艳御姐）

**基本信息：**
- 年龄：22 岁
- 身份：听雨楼少主
- 武功：轻功/暗器
- 定位：女主 1 号，入队

**外貌：**
- 一袭白衣，清冷如月
- 眉眼如画，气质高冷
- 腰间挂着听雨楼令牌

**性格：**
- 表面冷漠，内心柔软
- 聪明冷静，擅长谋略
- 对主角逐渐敞开心扉

**入队条件：**
- 主线剧情自动入队（第 1 章）

**攻略流程：**

| 阶段 | 好感度 | 触发事件 |
|------|--------|----------|
| 初识 | 0-20 | 主线相遇，她救了你 |
| 熟悉 | 20-40 | 一起执行任务，展现能力 |
| 信任 | 40-60 | 帮她解决听雨楼危机 |
| 心动 | 60-80 | 深夜谈心，她卸下防备 |
| 表白 | 80-100 | 选择表白，进入感情线 |

**关键事件：**
1. **第 2 章：** 她为你挡下一枚暗器（受伤）
2. **第 4 章：** 你帮她找回听雨楼失物（信任）
3. **第 6 章：** 她在你低谷时安慰你（心动）
4. **终章：** 根据好感度决定结局

**专属结局：**
- **好感度 100：** 与你归隐，成为你的贤内助
- **好感度 80-99：** 继续经营听雨楼，与你保持联系
- **好感度<80：** 无感情线，纯伙伴

**经典台词：**
- "别误会，我只是完成任务。"（初期）
- "你...要小心。"（中期）
- "如果有来生，我想做个普通人。"（后期）
- "你去哪，我去哪。"（结局）

### 2.3 多女主系统机制

**好感度系统：**

| 好感度 | 状态 | 解锁内容 |
|--------|------|----------|
| 0-19 | 陌生 | 基础对话 |
| 20-39 | 认识 | 解锁基础任务 |
| 40-59 | 熟悉 | 解锁支线任务 |
| 60-79 | 友好 | 解锁特殊对话 |
| 80-99 | 亲密 | 解锁感情线 |
| 100 | 生死之交 | 解锁专属结局 |

**多女主同时攻略：**
- ✅ 可以同时攻略多个女主
- ⚠️ 某些女主会吃醋（触发特殊事件）
- ⚠️ 终章需要选择（或全都要，隐藏结局）

**吃醋系统：**

```
示例：
苏婉儿好感度 80，药灵儿好感度 80

苏婉儿："你和药灵儿走得太近了。"
```
