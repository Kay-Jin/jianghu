## 场景 4：奇策制定

**画面：** 军事会议，众人围在地图前。

**【铁山】**
> 「兄弟，说实话——我们打不过。」
> 「一千五对十万，这不是打仗，这是送死。」

**【主角】**
> 「所以不能正面硬打。」
> 「我们要用奇策。」

**【作战计划 — 「火攻 + 伏击」】**
```
【第一阶段：诱敌深入】
  - 丐帮在前方骚扰，假装溃败
  - 引诱金国先锋进入伏击圈

【第二阶段：火攻】
  - 临渊城外有密林，适合火攻
  - 需要大量火油和干草
  - 听雨楼负责收集物资

【第三阶段：伏击】
  - 各门派高手埋伏在密林两侧
  - 金军进入火圈后，两面夹击

【第四阶段：斩首行动】
  - 主角 + 核心伙伴
  - 直取完颜宗望
  - 擒贼先擒王

风险：
  ⚠️ 如果火攻失败，联军将被包围
  ⚠️ 如果斩首行动失败，完颜宗望将反扑
  ⚠️ 需要精确的时间配合
```

## 场景 7：临渊城保卫战

**画面：** 次日清晨，金国大军出现在地平线上。

**【战场全景】**
```
金军：100,000 人（骑兵 + 步兵）
联军：约 1,500 人（各门派）

地形：临渊城外密林
天气：晴，风向东南（有利于火攻）
```

**【第一阶段：诱敌深入】**
```
丐帮三百帮众在前方骚扰。
假装不敌，向密林方向撤退。

金军先锋（5,000 骑兵）追击！

【紧张时刻】
如果诱敌失败 → 火攻无法实施 → 战斗进入困难模式
如果诱敌成功 → 金军进入伏击圈 → 火攻开始

结果：✅ 诱敌成功！
金军先锋进入密林！
```

**【第二阶段：火攻】**
```
「点火！」

密林四周同时起火！
火借风势，迅速蔓延！
金军骑兵陷入火海，马匹惊乱！

【火攻效果】
  - 金军先锋损失约 30%
  - 剩余部队混乱
  - 但主力尚未到达
```

**【第三阶段：伏击】**
```
各门派高手从密林两侧杀出！

铁血盟：正面冲锋！
少林寺：近战格斗！
华山派：剑法压制！
丐帮：骚扰后方！

【战斗高潮】
你与伙伴在敌阵中冲杀——

铁山：「杀——！！」
苏清商：「听雨剑法！」
药辛夷：「大哥！吃药！」（增益buff）
赵小满：「大哥！我发现完颜宗望了！」
林晚照：「在那边！」
```

**【第四阶段：斩首行动】**
```
主角 + 核心伙伴冲向完颜宗望！

完颜宗望（BOSS）：
  HP：极高
  攻击：极高
  防御：极高
  特殊技能：
    - 「金戈铁骑」（骑兵冲锋）
    - 「完颜霸体」（免疫控制 10 秒）
    - 「帝王之怒」（全屏伤害）

【战斗策略】
  1. 先清除护卫
  2. 用药辛夷的增益buff提升属性
  3. 林晚照用毒削弱BOSS
  4. 苏清商主攻
  5. 主角用「破军」+「山河社稷功」终结

⚠️ 战斗中可能出现：
  - 伙伴受伤（需要药辛夷治疗）
  - 金军援军（需要铁山挡住）
  - 天气变化（风向改变影响火攻）
```
