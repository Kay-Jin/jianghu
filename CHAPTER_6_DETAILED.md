**画面：** 主角独自北上，调查黄泉教总舵。

**【北方·黄泉教总舵外围】**

**【主角潜入】**
```

【发现】
- 黄泉教正在大规模集结人手
- 他们在准备一场大战
- 目标：……临渊城？
```

**【偶遇林晚照】**

**画面：** 黄泉教总舵花园，一个黑衣女子独自赏月。

**林晚照：** 「来了？」
**林晚照：** 「我等你很久了。」

**主角：** 「你知道我会来？」

**林晚照：** 「叶长风，武林大会的英雄。」
**林晚照：** 「你揪出了无痕哥哥，下一个目标当然是黄泉教。」
**林晚照：** 「但你找错人了。」

**【选择】**
```
A) "你什么意思？"（追问）
B) "你是敌人，少废话。"（敌对）
C) "我想听听你的说法。"（倾听）
```

**【选择 C — 倾听（推荐）】**

**林晚照：** 「你以为黄泉教是邪教？」
**林晚照：** 「也许吧。但你知道黄泉教为什么存在吗？」

**【林晚照讲述黄泉教的真相】**

**林晚照：** 「三十年前，黄泉教不是邪教。」
**林晚照：** 「它叫'正义盟'，是一群江湖人士为了对抗朝廷暴政而建立的。」
**林晚照：** 「我父亲是创始人之一。」
**林晚照：** 「蔡京也是。」

**主角：** 「蔡京也是创始人？」

**林晚照：** 「对。但他后来变了。」
**林晚照：** 「他当上宰相后，把黄泉教变成了自己的工具。」
**林晚照：** 「我父亲发现了他的真面目，想要揭发他。」
**林晚照：** 「然后……我父亲就'意外'死了。」

**【主角独白】**
> 「黄泉教原来是正义组织……」
> 「被蔡京从内部腐化了。」
> 「那药辛夷的娘……」

**林晚照：** 「你想知道药辛夷的娘是怎么死的？」
**林晚照：** 「不是我杀的。是蔡京的人。」
**林晚照：** 「因为她也在调查蔡京。」
**林晚照：** 「蔡京借刀杀人，让黄泉教背了黑锅。」

**【选择】**
```
A) "我相信你。但怎么证明？"（理性）
B) "你在为自己洗白。"（怀疑）
C) "如果你说的是真的，我们一起对付蔡京。"（合作）
```

**【选择 C — 合作（推荐）】**

**林晚照：** （沉默片刻）「……你真的相信我？」
**林晚照：** 「三年来，你是第一个愿意听我说话的人。」
**林晚照：** 「好。我给你证据。」

**【获得关键证据】**
```
📜 黄泉教密档（完整版）
  - 蔡京与黄泉教的关系
  - 蔡京引金国入关的计划
  - 武林理事会内奸名单
  - 黄泉教教主真实身份线索

🔑 黄泉教密道钥匙
  - 可以进入黄泉教核心区域

林晚照好感：+25
⚠️ 注意：与林晚照合作会影响药辛夷好感
```

**【蝴蝶效应】**
```
⚠️ 此处的选择将影响第 7 章和第 8 章：
```
