# 🗡️ 江湖打工人 - 六边形战棋 RPG

**Sprint 1 & 2 ✅ 完成** | Demo v0.1 可发布

## 🎮 项目简介

一款 Godot 4.3 引擎开发的六边形战棋 RPG 游戏，融合现代穿越元素与武侠世界观。

## ✨ 核心功能

### 战斗系统
- 六边形网格战棋
- 回合制战斗（移动→攻击/武功→结束回合）
- 移动/攻击范围可视化
- 12 种武功动画特效（剑气/龙形/毒雾/佛光等）
- 程序化音效（14 种战斗音效）

### 场景
- **主菜单** - 7 个功能按钮
- **序章教学** - 穿越醒来→对话→教学战斗
- **断剑山庄** - 首个城镇（9 个 NPC，铸剑主题）
- **世界大地图** - 7 座城镇，9 大区域
- **存档/读档** - JSON 格式，5 个槽位
- **好感度面板** - 6 级好感度 × 4 位女主

### 角色
- **叶长风** - 主角（现代装/战斗装/正装 3 套时装）
- **苏清商** - 听雨楼少主（冷艳御姐）
- **药辛夷** - 青囊谷传人（活泼萝莉）
- **赵小满** - 小乞丐（虎牙+比V）
- **林晚照** - 黄泉教圣女（忧郁深沉）
- **铁山** - 铁血盟骨干（凶悍大汉）
