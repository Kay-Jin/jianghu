## 一、命名原则

### 1.1 禁用清单（❌ 绝对不用）

**常见烂大街姓氏：**
- 林、萧、叶、楚、苏、陆、顾、沈（仙侠用烂了）

**常见烂大街名字：**
- 单字：逍、炎、凡、尘、逸、辰、风、云、天、神
- 双字：子轩、浩然、宇轩、梓轩、一鸣、天行

**其他游戏/小说用过的：**
- 萧炎、叶凡、林动、楚阳、苏铭...（玄幻小说主角名）
- 李逍遥、景天、云天河、百里屠苏...（游戏主角名）

**金庸古龙用过的武功：**
- 独孤九剑、降龙十八掌、六脉神剑、葵花宝典...
- 天外飞仙、小李飞刀、例无虚发...

### 1.2 鼓励方向（✅ 推荐）

**姓氏选择：**
- 中等频率姓氏：江、谢、傅、傅、晏、纪、许、程
- 复姓：皇甫、夏侯、诸葛、欧阳、司徒（但要用得巧妙）
- 自创姓：罕见但有韵味

**名字风格：**
- 从《诗经》《楚辞》取意象，但不直接用
- 生僻字但有寓意（如"翊""珩"" 璟"）
- 独特搭配，避免常见组合

**武功命名：**
- 描述意境，不直接抄
- 结合门派特色
- 有文化底蕴但不烂俗

## 六、招式命名规则

### 6.1 命名公式

**公式 A：意象 + 动作**
- 例：流云 + 出岫 = **流云出岫**
- 例：暴雨 + 倾盆 = **暴雨倾盆**

**公式 B：数字 + 意象**
- 例：七 + 星 = **七星剑诀**
- 例：九 + 转 = **九转轮回功**

**公式 C：动词 + 名词**
- 例：破 + 岳 = **破岳拳**
- 例：听 + 雨 = **听雨剑法**

### 6.2 禁用词（❌ 不用）

- 龙、凤、神、魔、天、地、玄、黄（太常见）
- 独孤、不败、无敌（太中二）
- 九阴、九阳、乾坤、太极（金庸用烂了）

### 6.3 鼓励词（✅ 推荐）

- 自然意象：云、雨、雪、霜、风、烟、波
- 时间意象：朝、暮、春、秋、晓、晚
- 情感意象：愁、思、念、怨、悔
- 动作意象：行、归、逝、返、溯

## 十、建筑命名规则 ⭐（Kay 强制要求）

### 10.1 核心原则

**建筑特点必须符合名字，不能随意取。**

这是 Kay 的强制要求，必须严格执行。

### 10.2 具体规则

| 建筑名 | 必须符合的特征 | 禁止 |
|--------|---------------|------|
| **断剑山庄** | 铸剑主题、剑形装饰、铁匠元素、断剑标志 | 不能出现药铺/青楼元素 |
| **听雨楼** | 雨景、情报/暗道、文人雅致、水榭楼台 | 不能出现铸剑炉/军营元素 |
| **青囊谷** | 药园、医馆、毒草、自然朴素 | 不能出现华丽装饰/武器 |
| **铁血盟** | 军事化、训练场、武器架、简朴实用 | 不能出现文雅/华丽元素 |
| **黄泉教** | 阴森、暗黑、毒雾、诡异装饰 | 不能出现明亮/温馨元素 |
| **临渊城** | 繁华、权力、鱼龙混杂、不夜城 | 不能出现偏僻/简朴元素 |
| **姑苏城** | 水乡、园林、精致、富庶文雅 | 不能出现军事/荒凉元素 |
| **燕山城** | 肃杀、军事、难民、忠义 | 不能出现繁华/文雅元素 |

### 10.3 检查清单

**每次设计建筑/城镇时必须检查：**
- [ ] 建筑外观是否符合名字含义？
- [ ] 内部装饰是否符合门派/势力特色？
- [ ] NPC 行为是否符合场景氛围？
- [ ] 是否有与名字矛盾的元素混入？
