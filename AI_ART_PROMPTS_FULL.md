# 六、批量生成建议

## 按优先级排序

| 优先级 | 角色 | 数量 | 说明 |
|--------|------|------|------|
| **P0 立刻生成** | 主角全5套外装 + 沈清鸢/白芷/钟灵 + 沈刃/顾昱明 + 贺云隐/义父 | 14张 | 游戏可玩的最低需求 |
| **P1 近期生成** | 霍毓桐/凌离雪/卫追云/珂星月 + 聂无殇/顾昱明/方烽 + 司空无迹/完颜宗望/蔡京 + 锤九黎/百晓生 | 16张 | 完整游戏体验 |
| **P2 后期填充** | 云墨烟/程烟笔/姜十三 + 罗衡/宁奔/巴图尔 + 其余反派/NPC | 30张 | 丰满内容 |
| **P3 最后完成** | 双生刺客/蛊婆/无相神僧 + 所有复用NPC原型 | 20张 | 完整美术 |

## 推荐工作流

```
1. 通义万相（免费）→ 出原型草稿，验证设计是否对味
2. Midjourney v7 → 精修主要角色（建议用 --sref 参数保持角色一致性）
3. Stable Diffusion + LoRA → 批量生产NPC变体（同一模型换装/换色）
4. 立绘完成后 → Meshy AI 上传图生成3D基础模型
5. Blender精修 → Mixamo绑骨 → 导入Unity
```

## 角色一致性技巧（Midjourney）

```
# 保持同一角色多表情一致：
/imagine 角色描述词 --sref [第一张成功图的URL] --sw 100

# 批量生成表情变体：
在提示词末尾加：
happy expression / angry expression / sad expression / combat stance
（保持所有其他词不变）
```
