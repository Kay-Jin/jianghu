## 一、核心设计理念

### 1.1 三大支柱

| 支柱 | 说明 |
|------|------|
| **开放式主线** | 非线性，多路径，自由选择探索顺序 |
| **限制变强** | 多维限制机制，避免一刀 999 |
| **探索驱动** | 隐藏要素、奇遇、探索奖励 |

### 1.2 玩家体验目标

```
自由度 ←→ 引导性
  ↑         ↑
  └────┬────┘
       ↓
   平衡点：玩家感觉自由，但不会迷路

挑战性 ←→ 成就感
  ↑         ↑
  └────┬────┘
       ↓
   平衡点：难但有办法，成长可见
```

## 二、开放世界主线结构（v2.2 新增）

### 2.1 设计理念（参考河洛群侠传 / 侠客风云传前传）

| 设计目标 | 说明 |
|---------|------|
| **无强制线性顺序** | 序章结束后，5 条主线全部解锁，玩家自由选择推进顺序 |
| **难度自然缩放** | 各主线 Boss 战难度与玩家等级动态匹配（±2 级范围） |
| **真结局条件** | 全部 5 条主线 + 24 个主线任务完成后，解锁终章「侠之大者」 |
| **隐形时间轴** | 游戏内有年历系统，部分支线/奇遇有时限；主线不限时 |
| **线路互相影响** | 某条主线的选择会影响其他主线的 NPC 态度/可选项 |

### 2.2 初始天赋（3 选 1）

| 天赋 | 效果 | 适合玩法 |
|------|------|----------|
| **悟性过人** | 武功领悟速度 +20% | 武功流 |
| **天生神力** | 初始攻击 +10%，内力 -10% | 暴力流 |
| **机敏聪慧** | 探索发现率 +20%，战斗闪避 +5% | 探索流 |
| **医术精通** | 初始医术 +20%，药品效果 +20% | 辅助流 |
| **交际达人** | NPC 好感度增长 +20% | 剧情流 |

### 2.3 初始属性

## 三、成长限制机制（详细）

### 3.1 内力系统

**设计目的：** 限制无限放技能

**机制：**

```
内力上限 = 基础内力 × (1 + 内功等级加成)

战斗内力消耗：
- 普通攻击：0
- 初级武功：5-10 点
- 中级武功：15-25 点
- 高级武功：30-50 点
- 绝学：50-100 点

内力恢复：
- 战斗外：每分钟恢复 10%
- 战斗中：每回合恢复 5 点
- 打坐：每小时恢复 50%（可加速）
- 丹药：瞬间恢复 20-50%
```

**策略影响：**
- 不能无限放最高伤害技能
- 需要合理分配内力
- 低级技能也有用（省内力）

### 3.2 武功领悟系统

**设计目的：** 获得秘籍≠学会，需要时间成本

**机制：**

```
领悟进度 = f(悟性，秘籍难度，指导，时间)

领悟方式：
1. 自行领悟
   - 需要：秘籍 + 时间（游戏内天数）
   - 公式：天数 = 秘籍难度 × 100 / 悟性
   
2. 名师指导
   - 需要：NPC 好感度≥60 + 银两
   - 效果：时间减少 50%
   
3. 闭关修炼
   - 需要：安全地点 + 充足补给
   - 效果：时间减少 30%，但无法做其他事
```

**领悟阶段：**

| 阶段 | 进度 | 效果 |
|------|------|------|
| 未入门 | 0% | 无法使用 |
| 初学乍练 | 1-30% | 威力 50%，消耗 120% |
| 驾轻就熟 | 31-60% | 威力 80%，消耗 100% |
| 融会贯通 | 61-90% | 威力 100%，消耗 90% |
| 出神入化 | 91-100% | 威力 120%，消耗 80% |
| 登峰造极 | 100% + 特殊条件 | 威力 150%，解锁特效 |

### 3.3 瓶颈系统

**设计目的：** 强制玩家探索，不能闭关到无敌

**大境界瓶颈：**

| 瓶颈 | 突破条件 | 预估时间 |
|------|----------|----------|
| 初窥→小成 | 战斗胜利 10 场 | 2-3 小时游戏 |
| 小成→贯通 | 领悟 3 本秘籍 + 名师指导 | 5-8 小时游戏 |
| 贯通→登堂 | 完成一条主线 + 声望 500 | 15-20 小时游戏 |
| 登堂→返璞 | 多结局条件，极难 | 30+ 小时游戏 |

**突破方式：**

```
瓶颈突破 = 条件满足 + 突破尝试

突破尝试：
- 成功率 = 基础 50% + 加成
- 加成来源：丹药（+10%）、高人指点（+20%）、心境（+10%）
- 失败惩罚：3 天内无法再次尝试
- 成功效果：境界提升，属性大幅提升
```

### 3.4 时间成本系统

**设计目的：** 修炼需要时间，错过事件

**游戏内时间机制：**

```
时间流逝：
- 移动：每格 5 分钟
- 战斗：每场 30 分钟
- 修炼：每小时计算
- 休息：6 小时/晚

事件触发：
- 主线任务：时间不敏感
- 支线任务：部分限时（3-7 天）
- 随机事件：时间敏感（错过无）
- 节日事件：特定日期
```

**玩家抉择：**
- 闭关修炼 30 天 → 可能错过限时任务
- 到处探索 → 修炼进度慢
- 平衡发展 → 最佳但最难

### 3.5 资源稀缺系统

| 资源 | 获取方式 | 稀缺性 | 用途 |
|------|----------|--------|------|
| **修炼丹** | 购买/炼制/掉落 | 中 | 加速领悟 |
| **突破丹** | 炼制/高难度掉落 | 高 | 提高突破成功率 |
| **内力丹** | 购买/炼制 | 低 | 恢复内力 |
| **秘籍** | 探索/任务/战斗 | 高 | 学习武功 |
| **银两** | 任务/战斗/经营 | 中 | 购买物品 |

**稀缺设计原则：**
- 高级资源不能无限获取
- 某些资源唯一（错过无）
- 鼓励探索和资源管理

### 3.6 武功冲突系统

**设计目的：** 需要取舍，不能全都要

**冲突规则：**

| 武功 A | 武功 B | 冲突类型 | 效果 |
|--------|--------|----------|------|
| 阳刚内功 | 阴柔内功 | 互斥 | 只能选其一 |
| 剑法专精 | 刀法专精 | 竞争 | 同时修炼效率 -50% |
| 正道武功 | 魔教武功 | 声望冲突 | 修炼后声望变化 |
| 速成功法 | 正统功法 | 代价 | 速成有副作用 |

**例子：**
- 《九阳神功》（阳刚）vs《九阴真经》（阴柔）→ 只能选一个
- 修炼魔教武功 → 正道 NPC 好感度下降

### 4.1 大地图结构

**区域划分：**

```
┌──────────────────────────────────────────────────────────┐
│  西北（荒漠）        北（河北）         东北（辽东）      │
│  - 龙门客栈          - 铁血盟           - 金国营地       │
│  - 莫高窟            - 燕山             - 长白山          │
│  - 丝绸之路          - 沧州             - 黑风寨          │
├──────────────────────────────────────────────────────────┤
│  西（蜀中）          中（汴梁）         东（沿海）        │
│  - 断剑山庄          - 京城             - 沿海帮派        │
│  - 峨眉              - 六扇门           - 蓬莱岛          │
│  - 青城山            - 天机阁           - 倭寇营地        │
├──────────────────────────────────────────────────────────┤
│  西南（岭南）        南（江南）         东南（东海）      │

│  - 十万大山          - 杭州             - 隐藏岛屿        │
└──────────────────────────────────────────────────────────┘
```

### 4.2 探索要素

**每个区域包含：**

| 要素 | 数量 | 说明 |
|------|------|------|
| 城镇 | 1-3 个 | 补给、任务、剧情 |
| 门派 | 0-2 个 | 拜师、学武 |
| 野外 | 若干 | 遇敌、采集、隐藏事件 |
| 副本 | 1-2 个 | 秘境、古墓、宝藏 |
| 隐藏地点 | 0-1 个 | 特殊条件解锁 |

### 4.3 探索驱动机制

**如何鼓励探索？**

| 机制 | 实现方式 | 奖励 |
|------|----------|------|
| **传闻系统** | NPC 对话透露线索 | 引导玩家 |
| **地图迷雾** | 未探索为黑色 | 收集欲 |
| **成就系统** | 探索度成就 | 称号/奖励 |
| **限时事件** | 某些事件限时 | 紧迫感 |
| **多周目** | 二周目新区域 | 重玩价值 |

**传闻系统示例：**

```
NPC 对话：
"听说西边的莫高窟最近有奇怪的光芒，可能是有秘籍出世..."
（解锁莫高窟副本标记）

"北边燕山有个隐居的老头，据说武功深不可测..."
（解锁隐藏 NPC，需要声望 300 才能拜访）

"江南听雨楼的密道，只有他们少主知道..."
（需要与听雨楼好感度 80+ 解锁）
```

### 4.4 奇遇系统

**奇遇类型：**

| 类型 | 触发条件 | 概率 | 奖励 |
|------|----------|------|------|
| **高人传功** | 探索隐藏地点 + 特定选择 | 1% | 直接获得武功 |
| **古墓宝藏** | 解谜 + 战斗 | 5% | 装备/秘籍 |
| **动物报恩** | 选择善良（不杀动物） | 3% | 特殊道具 |
| **时空错位** | 穿越者专属 | 2% | 现代物品 |
| **秘籍残页** | 探索任意地点 | 10% | 收集合成 |

**奇遇设计原则：**
- 不强制，纯探索奖励
- 有些是彩蛋（现代物品）
- 有些是实质性奖励（武功/装备）
- 概率低但值得探索

**奇遇示例：**

```
【时空错位】
你在野外探索时，突然感到一阵眩晕...
醒来发现地上有一个奇怪的金属盒子（手机？）
打开后只有一句话："记住，历史可以改变"
获得物品：【神秘手机】（无法使用，但偶尔会震动）

【动物报恩】
你遇到一只受伤的小白狐
选择：
A) 杀了它（获得狐皮，-声望）
B) 不管它（无变化）
C) 帮它治疗（消耗药品，+善良值）

选择 C 后，3 天后小白狐带到一个隐藏山洞
获得：【千年灵芝】×1
```

## 五、开放主线设计（详细）

### 5.1 三线结构

```
                    序章（教学）
                        ↓
                  第 1 章（开放起点）
                        ↓
              ┌─────────┼─────────┐
              ↓         ↓         ↓
           北线       东线       西线
        （家国）    （江湖）    （个人）
              ↓         ↓         ↓
              └─────────┼─────────┘
                        ↓
                  第 6 章（三线汇聚）
                        ↓
                  第 7 章（最终抉择）
                        ↓
                  第 8 章（终极对决）
                        ↓
                    终章（多结局）
```

### 5.2 三条主线详情

**北线（家国）：**

| 章节 | 剧情 | 关键选择 |
|------|------|----------|
| 第 2 章 | 遇到铁血盟，了解抗金形势 | 加入/不加入 |
| 第 3 章 | 参与抗金行动 | 激进/稳健 |
| 第 4 章 | 发现朝廷有内奸 | 揭发/隐忍 |
| 第 5 章 | 联合各方势力 | 外交/武力 |

**东线（江湖）：**

| 章节 | 剧情 | 关键选择 |
|------|------|----------|
| 第 2 章 | 卷入门派纷争 | 站队 A/站队 B/中立 |
| 第 3 章 | 参加武林大会 | 参赛/旁观 |
| 第 4 章 | 发现盟主阴谋 | 揭露/隐忍 |
| 第 5 章 | 江湖追杀 | 逃跑/反击 |

**西线（个人）：**

| 章节 | 剧情 | 关键选择 |
|------|------|----------|
| 第 2 章 | 义父揭露部分身世 | 相信/怀疑 |
| 第 3 章 | 寻找身世线索 | 探索/放弃 |
| 第 4 章 | 发现真相 | 接受/拒绝 |
| 第 5 章 | 面对仇人 | 复仇/宽恕 |

### 5.3 路线锁定机制

**软锁定（可改变）：**
- 选择站队 A，与 B 关系恶化，但可修复
- 需要付出代价（时间/资源/任务）

**硬锁定（不可改变）：**
- 杀死关键 NPC → 相关剧情永久关闭
- 某些选择后进入隐藏路线

**例子：**

```
【硬锁定示例】
第 3 章：发现盟主是反派
选择：
A) 当众揭露 → 进入"正义线"，与盟主为敌
B) 私下对峙 → 进入"谈判线"，可能策反
C) 保持沉默 → 进入"隐忍线"，后期背刺

选择后，其他路线永久关闭（二周目可体验）
```

### 5.4 多结局解锁条件

| 结局 | 基础条件 | 额外条件 |
|------|----------|----------|
| **隐退江湖** | 完成任意主线 | 好感度≥80 的 NPC≥3 人 |
| **为国捐躯** | 完成北线 | 选择死战 |
| **继续征程** | 完成任意主线 | 选择留下 |
| **返回现代** | 完成西线 | 收集 5 个穿越相关物品 |
| **破碎虚空** | 武道境界返璞归真 | 完成所有隐藏任务 |
| **一代宗师** | 完成所有门派支线 | 声望≥1000 |
| **富甲一方** | 完成所有经营支线 | 银两≥100 万 |
| **孤家寡人** | 队友全灭/背叛 | 自动触发 |

## 六、战斗系统（详细）

### 6.1 回合制战斗

**战斗流程：**

```
战斗开始
    ↓
确定行动顺序（身法决定）
    ↓
┌─────────────────────┐
│  玩家回合           │
│  - 攻击/武功/防御   │
│  - 物品/轻功/特殊   │
│  - 消耗内力         │
└─────────────────────┘
    ↓
敌方回合
    ↓
判断胜负
    ↓
胜利→掉落/经验    失败→读档/复活
```

### 6.2 行动选项

| 选项 | 说明 | 内力消耗 |
|------|------|----------|
| **普通攻击** | 基础伤害，不耗内力 | 0 |
| **武功** | 高伤害，耗内力 | 5-100 |
| **防御** | 减少 50% 伤害 | 0 |
| **物品** | 药品/暗器/特殊道具 | 0 |
| **轻功** | 闪避下次攻击/逃跑 | 10 |
| **蓄力** | 下回合伤害 +50% | 0 |
| **连携技** | 需要队友，高伤害 | 30-50 |

### 6.3 战斗策略

**属性克制：**

| 属性 | 克制 | 被克制 |
|------|------|--------|
| 阳刚 | 阴柔 | 玄妙 |
| 阴柔 | 玄妙 | 阳刚 |
| 玄妙 | 阳刚 | 阴柔 |

**战斗提示：**
- 观察敌人属性，选择克制武功
- 内力管理很重要
- 逃跑不可耻，有时是最佳选择

## 七、NPC 好感度系统

### 7.1 好感度等级

| 等级 | 数值 | 效果 |
|------|------|------|
| 陌生 | 0-19 | 基础对话 |
| 认识 | 20-39 | 解锁基础任务 |

## 九、多周目与终局玩法

### 9.1 核心设计理念

**"结局不是结束，是新的开始"**

参考：宝可梦/巫师 3/上古卷轴 5 的终局设计

**三大支柱：**
1. **通关后继续游玩** - 不强制进入二周目
2. **周期性活动** - 定期更新内容
3. **收集驱动** - 类似宝可梦的收集要素

### 9.2 通关后继续游玩（New Game+ 模式）

**玩家选择：**

```
通关后弹出：
┌─────────────────────────────────────────┐
│  恭喜完成游戏！                          │
│                                         │
│  请选择：                               │
│  A) 继续探索（保留当前状态）            │
│  B) 进入二周目（继承部分要素）          │
│  C) 两个存档都保留                      │
└─────────────────────────────────────────┘
```

**A) 继续探索模式：**

| 保留内容 | 说明 |
|----------|------|
| 所有武功 | 100% 保留 |
| 所有属性 | 100% 保留 |
| 所有物品 | 100% 保留 |
| 银两 | 100% 保留 |
| NPC 好感度 | 100% 保留 |
| 地图探索度 | 100% 保留 |

**解锁新内容：**
- 新支线任务（通关后专属）
- 新挑战 BOSS（隐藏高难度）
- 新区域（二周目专属区域）
- 新对话选项（根据结局变化）

**设计目的：**
- 玩家可以继续探索未完成的内容
- 不影响主线剧情（已完成的保持）
- 适合收集党/完美主义者

### 9.3 周期性活动系统

**设计理念：** 类似 MMO 的周常/月活动，但单机体验

| 活动 | 周期 | 内容 | 奖励 |
|------|------|------|------|
| **武林大会** | 游戏内每年 1 次 | 比武擂台，淘汰赛 | 声望/秘籍/装备 |
| **门派大比** | 游戏内每季 1 次 | 门派内部比武 | 门派贡献/武功 |
| **江湖悬赏** | 游戏内每月刷新 | 高难度任务 | 银两/稀有物品 |
| **秘境开启** | 游戏内特定日期 | 限时副本 | 唯一奖励 |
| **节日活动** | 现实节日对应 | 春节/中秋等特殊任务 | 限定物品 |

**武林大会（核心终局玩法）：**

```
参赛条件：
- 完成主线任意结局
- 声望≥500
- 推荐战力：通关后

赛制：
- 淘汰赛制（8 强/16 强/32 强）
- 随机匹配对手（包括新面孔）
- 每届对手强度提升

奖励：
- 冠军：【武林盟主】称号 + 唯一秘籍
- 亚军：稀有装备 + 大量银两
- 四强：中级奖励
- 参与奖：少量银两

特殊机制：
- 连续 3 届冠军→解锁隐藏 BOSS
- 累计参赛 10 次→解锁特殊对话
- 每届有新的参赛者（随机生成）
```

**江湖悬赏（周常刷新）：**

```
每月刷新 3-5 个高难度任务：

示例：
【追杀魔头】
- 目标：幽冥教护法
- 战力：推荐 5000
- 奖励：¥5000 + 【魔教秘籍】×1
- 限时：游戏内 30 天

【寻找珍宝】
- 目标：收集 5 个千年灵芝
- 奖励：¥3000 + 药王谷好感 +20
- 限时：游戏内 60 天

【护送任务】
- 目标：护送商队到汴梁
- 奖励：¥2000 + 随机装备
- 限时：游戏内 15 天
```

### 9.4 家园系统（玩家基地）

**设计理念：** 类似《巫师 3》的葡萄园/《上古卷轴 5》的房子

**家园功能：**

| 功能 | 说明 | 解锁条件 |
|------|------|----------|
| **修炼室** | 闭关修炼，效率 +50% | 通关后购买 |
| **练武场** | 练习武功，无消耗 | 初始 |
| **药圃** | 种植草药，自给自足 | 完成药王谷任务 |
| **藏经阁** | 展示收集的秘籍 | 收集 10 本秘籍 |
| **兵器架** | 展示收集的武器 | 收集 10 把武器 |
| **客房** | 邀请 NPC 做客 | 好感度≥80 |
| **仓库** | 存放多余物品 | 初始 |

**家园获取方式：**
- 通关后自动获得（根据结局不同，家园不同）
- 可升级（消耗银两/材料）
- 可装饰（家具/摆件）

**家园互动：**
- NPC 可来访（根据好感度）
- 特殊对话（家园专属）
- 家园成就（装饰度/收集度）

### 9.5 收集系统（宝可梦式驱动）

**收集要素：**

| 收集类型 | 数量 | 奖励 |
|----------|------|------|
| **武功秘籍** | 100 本 | 全收集解锁【武学宗师】成就 |
| **武器装备** | 80 把 | 全收集解锁【神兵利器】成就 |
| ** NPC 卡片** | 50 张 | 全收集解锁【江湖百晓生】成就 |
| **外观服饰** | 30 套 | 全收集解锁【时装达人】成就 |
| **成就奖杯** | 100 个 | 全收集解锁【完美江湖】成就 |
| **隐藏彩蛋** | 20 个 | 全收集解锁【穿越者】成就 |

**收集驱动机制：**

```
收集度显示：
- 秘籍：78/100
- 武器：45/80
- NPC 卡片：32/50
- 成就：65/100

收集提示：
- "听说西边的莫高窟有一本失传已久的秘籍..."
- "某位隐世高人手中似乎有一把神兵..."

收集奖励：
- 每收集 10% → 小奖励（银两/物品）
- 每收集 50% → 中奖励（稀有装备）
- 100% 收集 → 成就 + 唯一奖励
```

**NPC 卡片系统（类似宝可梦图鉴）：**

```
卡片获取：
- 与 NPC 首次对话→解锁基础卡片
- 好感度达到 60→解锁进阶卡片
- 好感度达到 100→解锁特殊卡片
- 完成 NPC 支线→解锁剧情卡片

卡片信息：
- 姓名/年龄/门派
- 性格/喜好/背景故事
- 武功/战绩/关系网
- 收集度（1-5 星）

卡片奖励：
- 收集同一门派全套→门派奖励
- 收集全部→隐藏成就
```

### 9.6 挑战系统（高难度内容）

**挑战类型：**

| 挑战 | 说明 | 奖励 |
|------|------|------|
| **试炼塔** | 100 层爬塔，每层更强 | 层数奖励 |
| **BOSS RUSH** | 连续挑战所有 BOSS | 时间奖励 |
| **无伤挑战** | 无伤通关特定 BOSS | 成就奖励 |
| **速通挑战** | 最快时间通关 | 排行榜 |
| **限制挑战** | 限制武功/装备通关 | 特殊奖励 |

**试炼塔（核心挑战）：**

```
层数：100 层

机制：
- 每层 1 个敌人（强度递增）
- 每 10 层 1 个精英 BOSS
- 每 50 层 1 个特殊机制层
- 100 层：隐藏真 BOSS

奖励：
- 每层：银两 + 随机物品
- 每 10 层：稀有装备
- 每 50 层：唯一秘籍
- 100 层：【天下第一】称号 + 隐藏结局

特殊机制层示例：
- 第 50 层：只能使用轻功
- 第 75 层：内力不恢复
- 第 90 层：敌人属性翻倍
```

### 9.7 动态世界系统

**设计理念：** 世界不静止，NPC 有自己的生活

**动态事件：**

| 事件类型 | 说明 | 触发条件 |
|----------|------|----------|
| ** NPC 游历** | NPC 离开原位置，到处走 | 好感度≥60 |
| **门派变迁** | 门派兴衰，掌门更替 | 游戏内时间 |
| **江湖传闻** | 新事件/新人物出现 | 随机 |
| **季节变化** | 春夏秋冬，场景变化 | 游戏内时间 |
| **节日庆典** | 春节/中秋等特殊活动 | 现实时间对应 |

**NPC 游历系统：**

```
好感度≥60 的 NPC 会：
- 偶尔离开原位置
- 到其他地方游历
- 给你寄信（分享见闻）
- 带回礼物（特产/物品）

示例：
【沈清鸢的信】
"我在江南遇到一个奇怪的说书人，
他讲的故事里竟然有你的名字...
附：苏州特产糕点×1"
```
