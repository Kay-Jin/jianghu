# 第一部分：项目概述

## 1.1 核心定位

| 维度 | 设定 |
|------|------|
| 游戏名称 | 江湖打工人（暂定，可改） |
| 游戏类型 | 2D 回合制 RPG |
| 参考作品 | 古龙群侠传 / 侠客风云传 / 古龙风云录 |
| 美术风格 | 2D 写实国风 |
| 叙事风格 | 金庸的宏大 + 古龙的写意 + 黄易的玄妙 |
| 目标平台 | Steam（PC） |
| 定价策略 | ¥48-68 买断制 |
| 团队规模 | 1 人（Kay）+ AI 助理 |

## 1.2 差异化卖点

1. **穿越者主角** — 现代项目经理穿越到古代，有梗有吐槽
2. **武功特效分级** — Lv.0-Lv.5，成长可见，一眼看出强弱
3. **原创命名体系** — 全部原创，不抄金庸古龙
4. **多女主吃醋** — 同时攻略触发吃醋事件，更真实
5. **天赋书系统** — 先天 + 后天 + 隐藏，可更换，有策略
6. **属性互相制约** — 属性互斥，不能全都要，必须取舍
7. **五条主线** — 北线家国 / 东线江湖 / 西线身世 / 第4线回现代 / 第5线家园
8. **终局玩法** — 通关后继续游玩，武林大会/悬赏/家园/试炼塔

| 维度 | 描述 |

# 第二部分：世界观设定

## 2.1 时代背景

**朝代：** 架空宋朝（类似北宋末年）  
**时间：** 宣和年间（约公元 1120 年）

**历史背景：**
- 北方：金国崛起，辽国衰落
- 中原：宋朝积弱，奸臣当道
- 江湖：武林盟式微，群雄并起
- 民间：百姓困苦，义军暗涌

**关键历史事件：**
- 1 年前：海上之盟（宋金联合灭辽）
- 现在：金国开始南下侵宋
- 未来（游戏结局时）：靖康之耻前夕

## 3.2 六位属性系统

**核心原则：** 属性互相制约，不能全都要

| 属性 | 影响 | 制约关系 | 初始范围 |
|------|------|----------|----------|
| **根骨** | 生命值、防御 | ↑根骨 → ↓身法 | 5-15 |
| **内力** | 内力上限、恢复 | ↑内力 → ↓气血上限 | 5-15 |
| **悟性** | 领悟速度、暴击 | ↑悟性 → ↓根骨 | 5-15 |
| **身法** | 速度、闪避 | ↑身法 → ↓防御 | 5-15 |
| **力量** | 攻击、负重 | ↑力量 → ↓身法 | 5-15 |
| **外貌** | NPC 初始好感 | ↑外貌 → ↓内力 | 5-15 |

**属性上限：**
- 单项属性上限：100
- 属性总和上限：300
- 突破上限需要特殊条件（天赋书/奇遇）

## 5.10 武学结构

**一门武学 = 多个招式（3-7 个）+ 被动技能（0-2 个）**

**招式类型（6 种）：**
1. **伤害型** — 直接造成伤害
2. **BUFF 型** — 给自己加增益
3. **DEBUFF 型** — 给敌人加减益
4. **多段型** — 多次伤害
5. **防御型** — 减少伤害/格挡
6. **特殊型** — 特殊效果（吸血/闪避/反击）

## 5.11 武学示例：《流云七剑》

**基本信息：** 剑法 | 阴柔属性 | 7 招 | 1 被动

| 登堂→返璞 | 多结局条件，极难 | 30+ 小时游戏 |

## 12.3 武功冲突

| 冲突类型 | 说明 | 示例 |
|----------|------|------|
| 正邪冲突 | 阳刚 vs 阴柔 | 溯光诀 vs 黄泉鬼录 → 只能选其一 |
| 属性冲突 | 不同属性效率降低 | 断剑心法 vs 青囊真解 → 效率 -50% |
