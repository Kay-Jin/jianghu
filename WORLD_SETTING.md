## 一、时代背景

### 1.1 历史框架

**朝代：** 架空宋朝（类似北宋末年）

**时间：** 宣和年间（约公元 1120 年）

**历史背景：**
- 北方：金国崛起，辽国衰落
- 中原：宋朝积弱，奸臣当道（蔡京、童贯等）
- 江湖：武林盟式微，群雄并起
- 民间：百姓困苦，义军暗涌

**关键历史事件（游戏时间线）：**
- 1 年前：海上之盟（宋金联合灭辽）
- 现在：金国开始南下侵宋
- 未来（游戏结局时）：靖康之耻前夕

### 1.2 世界观基调

**金庸的宏大：**
- 家国情怀贯穿主线
- 江湖与朝堂交织
- 个人命运与时代洪流

**古龙的写意：**
- 人物塑造重于招式
- 悬念与反转
- 经典台词风格

**黄易的玄妙：**
- 武道哲学（武道通玄）
- 精神境界描写
- 天人合一的武学理念

## 三、主角设定

### 3.1 基本信息

**姓名：** **玩家自定义**（游戏开始时输入，2-6 字）

**昵称/江湖称号：** 随剧情发展获得（如"穿越大侠""吐槽剑客""代码剑仙"等）

**年龄：** 28 岁（现代）→ 18 岁（穿越后身体）

**出身：** **程序员**（现代社畜，加班猝死后穿越）

**穿越方式：** **随机**（增加重玩价值，每次开局可能不同）

**穿越时间：** 游戏开始前 3 个月

**初始属性：** **随机生成**（根据穿越方式有差异）

### 3.2 性格特点

**核心性格：**
- 幽默乐观（现代人的吐槽能力）
- 有底线（不做违背良心的事）
- 重情义（对朋友/爱人忠诚）
- 有智慧（用现代思维解决古代问题）

**口头禅/特点：**
- 内心 OS 丰富（吐槽担当）
- 偶尔蹦出现代词汇（"内卷""996""躺平"）
- 对古代规矩不适应（制造笑点）
- 关键时刻靠得住（侠义内核）

### 3.3 成长线

```
第 1 章：穿越醒来，一脸懵逼
        ↓
第 2 章：被卷入江湖纷争，被迫学武
        ↓
第 3 章：结识伙伴，组建小队
        ↓
第 4 章：发现身世之谜（原主身份）
        ↓
第 5 章：江湖成名，被各方拉拢
        ↓
第 6 章：金国南下，面临抉择
        ↓
第 7 章：终极对决，为国为民
        ↓
终章：多结局（隐退/殉国/归隐等）
```

### 3.4 特殊能力（穿越者优势）

| 能力 | 说明 | 游戏内效果 |
|------|------|------------|
| **现代知识** | 知道历史走向 | 可预判某些事件 |
| **科学思维** | 用科学解释武功 | 悟性更高，创新招式 |
| **历史剧经验** | 看过很多武侠剧 | 识别套路，避免踩坑 |
| **吐槽能力** | 现代人的幽默感 | 对话选项更有趣 |

## 五、武学体系

### 5.0 核心设计理念

**开放式成长 + 平衡限制**

| 原则 | 说明 |
|------|------|
| **开放式主线** | 非线性，多路径，自由选择 |
| **限制变强** | 资源稀缺，时间成本，瓶颈机制 |
| **探索驱动** | 隐藏要素，奇遇，探索奖励 |
| **选择有意义** | 不同选择影响剧情/武功/结局 |

### 5.1 武功分类

| 类型 | 说明 | 例子 |
|------|------|------|
| **内功** | 修炼内力，提升根基 | 《九转玄功》《先天功》 |
| **外功** | 拳脚刀剑，招式技巧 | 《断剑剑法》《铁血刀法》 |
| **轻功** | 身法移动，逃跑追击 | 《燕子飞》《踏雪无痕》 |
| **暗器** | 远程攻击，偷袭 | 《飞星诀》《暴雨梨花针》 |
| **奇门** | 毒术/医术/阵法 | 《药王真经》《天机阵》 |

### 5.2 武功获取方式

| 方式 | 说明 |
|------|------|
| 门派学习 | 拜师后传授 |
| 秘籍获取 | 探索/战斗/任务获得 |
| 自创武功 | 后期融合多门武功自创 |
| NPC 传授 | 好感度高后传授 |
| 奇遇 | 特殊事件获得 |

### 5.3 战斗系统核心

**回合制战斗：**

```
行动顺序 = 速度属性决定

每回合可选：
- 攻击（普通攻击，不耗内力）
- 武功（消耗内力，伤害高）
- 防御（减少伤害）
- 物品（药品/暗器）
- 轻功（闪避/逃跑）
- 特殊（连携技/必杀技）
```

**内力系统：**
- 内力上限 = 内功等级决定
- 释放武功消耗内力
- 战斗外可打坐恢复
- 内力耗尽无法使用武功

## 六、开放式主线设计

### 6.1 非线性的核心结构

**传统线性（❌ 不采用）：**
```
序章→第 1 章→第 2 章→第 3 章→...→结局
```

**开放式主线（✅ 采用）：**
```
                    ┌→ 北线（抗金义军）→┐
                    │                   │
序章→自由探索→中→东线（江湖恩怨）→中→结局分支
                    │                   │
                    └→ 西线（身世之谜）→┘
```

**核心设计理念：**
- 序章后进入**大地图自由探索**
- 主线任务分散在各地，触发顺序由玩家决定
- 每条线有独立剧情，但最终汇聚到终局
- 某些选择会**永久关闭**其他路线（增加重玩价值）

### 6.2 三条主线

| 路线 | 主题 | 关键 NPC | 奖励 |
|------|------|----------|------|
| **北线（家国）** | 抗金救国 | 铁血盟、朝廷 | 声望、官职、群体战斗 |
| **东线（江湖）** | 门派恩怨 | 各门派、武林盟 | 武功秘籍、江湖名声 |
| **西线（个人）** | 身世之谜 | 义父、神秘人 | 家传武功、真相 |

**玩家选择影响：**
- 先走哪条线，后续剧情会不同
- 某些路线只能二选一（如帮 A 门派就不能帮 B 门派）
- 最终结局根据路线完成度解锁

### 6.3 支线系统

**支线类型：**

| 类型 | 说明 | 例子 |
|------|------|------|
| **伙伴支线** | 每个伙伴有独立任务线 | 帮苏婉儿找回听雨楼失物 |
| **门派支线** | 各门派委托任务 | 帮药王谷寻找草药 |
| **随机事件** | 探索中触发 | 路见不平拔刀相助 |
| **隐藏支线** | 特殊条件触发 | 连续 3 次选择善良选项 |

**支线奖励：**
- 武功秘籍
- 装备/物品
- 好感度
- 解锁隐藏结局

## 七、成长限制机制

### 7.1 为什么限制变强？

**问题：** 传统 RPG 玩家容易一刀 999，失去挑战

**我们的方案：** 多维限制，平衡成长

### 7.2 限制机制

| 机制 | 说明 | 效果 |
|------|------|------|
| **内力上限** | 内力 = 内功等级决定，无法无限放技能 | 战斗需要策略 |
| **武功领悟** | 秘籍获得≠学会，需要领悟时间 | 不能速成 |
| **瓶颈系统** | 每提升一个大境界需要突破 | 卡进度，强制探索 |
| **时间成本** | 修炼需要游戏内时间 | 不能闭关到无敌 |
| **资源稀缺** | 高级丹药/材料有限 | 不能无限嗑药 |
| **武功冲突** | 某些武功不能同时修炼 | 需要取舍 |

### 7.3 瓶颈突破设计

**武道境界突破条件：**

| 境界 | 突破条件 |
|------|----------|
| 初窥→小成 | 战斗经验积累（打赢 10 场） |
| 小成→贯通 | 领悟一本秘籍 + 名师指导 |
| 贯通→登堂 | 完成一条主线 + 声望达到 |
| 登堂→返璞 | 多结局条件之一，极难 |

**瓶颈突破方式：**
- 寻找高人指点（NPC 好感度）
- 完成特定任务（获得感悟）
- 达到隐藏条件（如不杀一人通关）

### 7.4 资源限制

| 资源 | 获取方式 | 限制 |
|------|----------|------|
| **修炼时间** | 客栈休息/闭关 | 游戏内时间流逝，错过事件 |
| **丹药** | 购买/炼制/掉落 | 高级丹药稀缺 |
| **秘籍** | 探索/任务/战斗 | 某些唯一，错过无 |
| **银两** | 任务/战斗/经营 | 前期紧缺 |

## 八、探索机制设计

### 8.1 大地图探索

**地图结构：**

```
┌────────────────────────────────────────────────────┐
│  西北（荒漠）    北（河北）    东北（辽东）          │
│     ↓              ↓              ↓               │
│  西（蜀中）    中（汴梁）    东（沿海）             │
│     ↓              ↓              ↓               │
│  西南（岭南）    南（江南）    东南（东海）         │
└────────────────────────────────────────────────────┘
```

**探索要素：**

| 要素 | 说明 | 奖励 |
|------|------|------|
| **城镇** | 补给、接任务、触发剧情 | 基础功能 |
| **野外** | 随机遇敌、采集、隐藏事件 | 资源、奇遇 |
| **门派** | 拜师、学武、支线任务 | 武功、人脉 |
| **副本** | 秘境、古墓、宝藏 | 高级装备、秘籍 |
| **隐藏地点** | 特殊条件解锁 | 唯一奖励 |

### 8.2 探索驱动机制

**如何鼓励探索？**

| 机制 | 说明 |
|------|------|
| **传闻系统** | NPC 对话透露隐藏地点线索 |
| **地图迷雾** | 未探索区域为黑色，探索后点亮 |
| **成就奖励** | 探索度达到 X% 解锁成就/奖励 |
| **限时事件** | 某些事件只在特定时间出现 |
| **多周目要素** | 某些地点二周目才开放 |

### 8.3 奇遇系统

**奇遇类型：**

| 类型 | 触发条件 | 奖励 |
|------|----------|------|
| **高人传功** | 探索隐藏地点 + 特定选择 | 直接获得武功 |
| **古墓宝藏** | 解谜 + 战斗 | 装备/秘籍 |
| **动物报恩** | 选择善良（不杀某些动物） | 特殊道具 |
| **时空错位** | 穿越者专属 | 现代物品乱入 |

**奇遇设计原则：**
- 不强制，纯探索奖励
- 有些是彩蛋（现代物品）
- 有些是实质性奖励（武功/装备）

## 九、剧情大纲（开放式结构）

### 序章：穿越醒来（教学关）

**剧情：**
- 主角在现代加班/意外 → 穿越到古代
- 醒来发现自己在荒郊野外
- 遇到第一个 NPC（引导角色）
- 学习基础操作（移动/对话/战斗）
- **自定义姓名/职业/穿越方式**

**目标：** 教学 + 引入世界观 + 角色创建

---

### 第 1 章：初入江湖（开放起点）

**剧情：**
- 被路过的门派弟子所救（或选择独自离开）
- 得知原主身份（某门派弟子/世家子弟）
- 决定暂时隐瞒穿越身份
- 第一次接触江湖纷争

**关键事件：**
- 拜师/入门（可选：7 大门派任选其一）
- 学习第一门武功
- 结识第一个伙伴（可选：3 人初始选 1 人）

**开放设计：**
- 门派选择不影响主线，但影响可用武功
- 伙伴选择影响后续支线
- 可拒绝拜师，走散人路线（更难）

---

### 第 2 章：门派风云（多路径）

**剧情：**
- 门派内部矛盾（师兄弟争斗）
- 发现有人勾结外敌
- 主角被卷入阴谋
- 第一次生死战斗

**关键事件：**
- 揭露内奸
- 获得第一本秘籍
- 伙伴 +1

**开放设计：**
- 可选择站队（大师兄 or 二师兄）
- 不同选择影响后续剧情
- 可选择不揭露（隐藏线）

---

### 第 3 章：江湖恩怨（大地图解锁）

**剧情：**
- 离开门派，游历江湖
- 遇到各门派人物
- 卷入多势力纷争
- 发现更大的阴谋

**关键事件：**
- 结识核心伙伴
- 武功小成
- 名声初显

**开放设计：**
- **大地图完全解锁**，可自由探索
- 三条主线开放（北线/东线/西线），顺序自选
- 某些地区需要声望/实力解锁

---

### 第 4 章：身世之谜（西线专属）

**剧情：**
- 发现原主真实身份（重要人物后代）
- 义父揭露部分真相
- 主角面临身份认同危机
- "我是谁？现代人还是古代人？"

**关键事件：**
- 身世揭秘（部分）
- 获得家传武功
- 内心成长

**开放设计：**
- 仅当探索西线时触发
- 可选择是否接受身份（影响后续）
- 拒绝身份可走隐藏路线

---

### 第 5 章：武林大会（东线专属）

**剧情：**
- 各门派汇聚，推选武林盟主
- 主角代表门派参赛（或散人身份）
- 发现盟主候选人有问题
- 揭露阴谋，一战成名

**关键事件：**
- 比武大会（多场战斗）
- 名声大噪
- 成为各方拉拢对象

**开放设计：**
- 仅当探索东线且声望足够时触发
- 可选择参赛 or 旁观
- 不同选择影响江湖声望

---

### 第 6 章：阴谋浮现（三线汇聚）

**剧情：**
- 发现武林盟主勾结金国
- 朝廷奸臣推波助澜
- 伙伴被抓/被杀（牺牲，取决于选择）
- 主角陷入低谷

**关键事件：**
- 重大挫折
- 伙伴牺牲（情感高潮，可避免）
- 主角成长（从搞笑到沉稳）

**开放设计：**
- 三条主线完成度影响此章剧情
- 伙伴牺牲可避免（需要前置条件）
- 不同选择影响最终结局分支

---

### 第 7 章：生死抉择（北线专属）

**剧情：**
- 金国大军南下
- 江湖人士面临选择（战 or 逃）
- 主角做出决定（为国为民）
- 联合各方势力，准备决战

**关键事件：**
- 组建联军
- 最终决战准备
- 主角武道突破

**开放设计：**
- 仅当探索北线时触发完整剧情
- 可选择战/逃/中立（影响结局）
- 联军规模取决于前置任务完成度

---

### 第 8 章：终极对决（最终章前奏）

**剧情：**
- 与最终反派决战
- 揭示反派动机（有深度）
- 主角获胜，但付出代价
- 金国退兵/宋朝暂保

**关键事件：**
- 最终 BOSS 战
- 反派临终遗言
- 主角成为大侠

**开放设计：**
- BOSS 战难度取决于前置准备
- 可寻求盟友协助（取决于好感度）
- 某些盟友可能牺牲（永久死亡）

---

### 终章：侠之大者（多结局）

**结局分支（根据玩家选择 + 完成度）：**

| 结局 | 条件 | 描述 |
|------|------|------|
| **A) 隐退江湖** | 好感度高，选择归隐 | 与爱人/伙伴隐居，不问世事 |
| **B) 为国捐躯** | 选择死战 | 战死沙场，成为民族英雄 |
| **C) 继续征程** | 选择留下 | 继续守护江湖，开放结局 |
| **D) 返回现代** | 特殊条件（穿越者专属） | 找到方法回到现代，但失去一切 |
| **E) 破碎虚空** | 武道满级 + 隐藏条件 | 达到传说境界，飞升而去 |
| **F) 一代宗师** | 完成所有门派支线 | 开宗立派，传承武学 |
| **G) 富甲一方** | 完成所有经营支线 | 成为首富，江湖传说 |
| **H) 孤家寡人** | 队友全灭/背叛 | 孤独终老，悲剧结局 |

**结局解锁机制：**
- 基础结局 3 个（A/B/C）- 任何玩家可达
- 特殊结局 3 个（D/E/F）- 需要特定条件
- 隐藏结局 2 个（G/H）- 极端路线

**多周目设计：**
- 一周目后解锁"继承模式"（保留部分武功）
- 二周目可探索新区域
- 三周目解锁真结局

## 十、幽默元素设计

### 10.1 幽默来源

| 类型 | 例子 |
|------|------|
| **现代词汇乱入** | "这个 boss 太难了，能不能氪金？" |
| **内心 OS 吐槽** | （这剧情我熟啊，电视剧里都是这么演的） |
| **古代人不理解** | "996 是什么？为何听起来很可怕？" |
| **反差萌** | 严肃场景突然搞笑 |
| **NPC 反应** | 古代人对主角奇葩言行的困惑 |

### 10.2 幽默边界

**可以搞笑：**
- 日常对话
- 伙伴互动
- 无关紧要的场景

**不能搞笑：**
- 伙伴牺牲场景
- 国家存亡抉择
- 最终决战
- 侠义精神表达

**原则：** 70% 幽默 + 30% 沉重，关键时刻必须严肃

## 十一、游戏主题

### 11.1 核心主题

**"侠之大者，为国为民"**

- 个人恩怨 vs 家国大义
- 逃避 vs 担当
- 现代思维 vs 古代责任

### 11.2 想传达的价值观

1. **侠义精神** - 不是打打杀杀，是责任与担当
2. **家国情怀** - 个人命运与国家命运相连
3. **友情爱情** - 珍惜身边的人
4. **成长** - 从普通人到大侠的蜕变

### 11.3 玩家体验目标

| 体验 | 目标 |
|------|------|
| 剧情 | 有笑有泪，结局感动 |
| 战斗 | 有策略，不无脑 |
| 养成 | 有成就感，成长可见 |
| 选择 | 有意义，影响结局 |
| 整体 | 玩完后有回味 |
