## 一、武功特效分级系统（核心！）

### 1.1 设计理念

**"成长可见，一眼看出强弱"**

玩家释放武功时，特效根据领悟阶段自动变化：
- 低等级：简单光效
- 高等级：华丽特效 + 环境互动

### 1.2 特效等级对照表

| 领悟阶段 | 特效等级 | 光晕强度 | 粒子数量 | 特殊效果 |
|----------|----------|----------|----------|----------|
| **未入门** | Lv.0 | 无 | 0 | 无 |
| **初学乍练** | Lv.1 | 微弱 (30%) | 10-20 | 基础光晕 |
| **驾轻就熟** | Lv.2 | 明显 (60%) | 30-50 | 光晕 + 轨迹 |
| **融会贯通** | Lv.3 | 强烈 (100%) | 60-100 | 光晕 + 轨迹 + 残影 |
| **出神入化** | Lv.4 | 华丽 (150%) | 120-180 | 以上 + 环境互动 |
| **登峰造极** | Lv.5 | 极致 (200%) | 200-300 | 以上 + 专属 BGM+ 全屏震动 |

### 1.4 特效元素详解

**光晕效果（Glow）：**
```
Lv.1: 武器/手掌边缘微弱发光
Lv.2: 明显发光，范围扩大
Lv.3: 强烈发光，有脉动感
Lv.4: 华丽发光，有色彩变化
Lv.5: 极致发光，有光环扩散
```

**粒子效果（Particles）：**
```
Lv.1: 少量粒子，沿攻击轨迹
Lv.2: 中等粒子，有拖尾
Lv.3: 大量粒子，有漩涡效果
Lv.4: 粒子爆发，有元素特效（火/水/风）
Lv.5: 粒子风暴，有元素实体化（火龙/水凤）
```

**残影效果（Afterimage）：**
```
Lv.1: 无
Lv.2: 无
Lv.3: 攻击时留下 1-2 个残影
Lv.4: 留下 3-5 个残影，残影有颜色
Lv.5: 留下 5-8 个残影，残影可独立攻击
```

**环境互动（Environment）：**
```
Lv.1-2: 无
Lv.3: 轻微空气波动
Lv.4: 明显环境效果（雪花/落叶/水波）
Lv.5: 强烈环境效果（暴风雪/落叶风暴/水龙卷）
```

**地面痕迹（Ground Mark）：**
```
Lv.1-2: 无
Lv.3: 短暂痕迹（1-2 秒消失）
Lv.4: 持久痕迹（5-10 秒消失）
Lv.5: 永久痕迹（直到离开场景）
```

**音效升级（Sound）：**
```
Lv.1: 基础破空声
Lv.2: 清晰破空声
Lv.3: 破空声 + 回音
Lv.4: 破空声 + 回音 + 环境音（雷声/风声）
Lv.5: 专属 BGM 片段（3-5 秒）+ 全屏震动
```

### 1.6 特效实现方案（Godot）

**节点结构：**
```
Character2D
├── Sprite
├── MartialArtsEffect (自定义节点)
│   ├── GlowSprite (光晕)
│   ├── ParticleSystem2D (粒子)
│   ├── AfterimageSprite (残影)
│   └── AudioStreamPlayer2D (音效)
└── AnimationPlayer
```

**核心代码：**
```gdscript
class_name MartialArtsEffect
extends Node2D

@export var effect_level: int = 0  # 0-5
@export var element_type: String = "cloud"  # cloud/sword/rain/blood/dream
@export var base_color: Color = Color(0.5, 0.7, 1.0)

var glow_sprite: Sprite2D
var particles: GPUParticles2D
var afterimage_timer: Timer

func _ready():
    setup_glow()
    setup_particles()
    setup_afterimage()

func play_effect(position: Vector2, direction: Vector2):
    match effect_level:
        0: pass  # 无特效
        1: play_level_1(position, direction)
        2: play_level_2(position, direction)
        3: play_level_3(position, direction)
        4: play_level_4(position, direction)
        5: play_level_5(position, direction)

func play_level_1(position: Vector2, direction: Vector2):
    # 微弱光晕
    glow_sprite.modulate = base_color * 0.3
    glow_sprite.scale = Vector2(0.5, 0.5)
    play_sound("basic_swoosh")

func play_level_2(position: Vector2, direction: Vector2):
    # 明显光晕 + 粒子拖尾
    glow_sprite.modulate = base_color * 0.6
    glow_sprite.scale = Vector2(0.8, 0.8)
    emit_particles(position, direction, 30)
    play_sound("clear_swoosh")

func play_level_3(position: Vector2, direction: Vector2):
    # 强烈光晕 + 大量粒子 + 残影
    glow_sprite.modulate = base_color * 1.0
    glow_sprite.scale = Vector2(1.0, 1.0)
    emit_particles(position, direction, 80)
    create_afterimage(position, 2)
    play_sound("swoosh_with_echo")

func play_level_4(position: Vector2, direction: Vector2):
    # 华丽光晕 + 粒子风暴 + 多残影 + 环境互动
    glow_sprite.modulate = base_color * 1.5
    glow_sprite.scale = Vector2(1.3, 1.3)
    emit_particles(position, direction, 150)
    create_afterimage(position, 5)
    create_environment_effect(element_type)
    create_ground_mark(position, 8.0)
    play_sound("epic_swoosh_with_ambience")

func play_level_5(position: Vector2, direction: Vector2):
    # 极致光晕 + 粒子实体化 + 残影攻击 + 环境风暴 + 专属 BGM
    glow_sprite.modulate = base_color * 2.0
    glow_sprite.scale = Vector2(1.8, 1.8)
    emit_particles(position, direction, 250, true)  # true = 实体化
    create_afterimage(position, 8, true)  # 残影可攻击
    create_environment_storm(element_type)
    create_ground_mark(position, -1.0)  # -1 = 永久
    play_custom_bgm("ultimate_theme")
    screen_shake(0.5)  # 全屏震动 0.5 秒
```
