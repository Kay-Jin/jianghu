# 第一部分：城镇设计原则

## 1.1 设计理念

**真实历史城镇 + 虚构江湖元素 = 沉浸感最大化**

- 城镇名称：使用真实宋朝地名
- 城镇特色：结合真实历史特点
- 江湖元素：虚构门派/组织融入真实城镇
- 历史事件：结合真实历史背景（靖康之耻前夕）

## 1.2 宋朝历史背景对照

| 真实历史 | 游戏中对应 |
|----------|------------|
| 北宋末年（宣和年间，约 1120 年） | 游戏时间线 |
| 金国崛起，辽国衰落 | 北方威胁 |
| 蔡京、童贯等奸臣当道 | 朝廷反派 |
| 海上之盟（宋金联合灭辽） | 1 年前历史事件 |
| 靖康之耻前夕 | 游戏结局时间 |

## 🏯 1. 开封府（东京汴梁）— 北宋都城

**历史地位：** 北宋都城，当时世界最大城市，人口 150 万+  
**游戏定位：** 全国政治中心，朝廷与江湖交汇之地

**历史特色：** 清明上河图原型城市，繁华至极

| **黑市** | 虚构 | 稀有物品/暗线任务 |
