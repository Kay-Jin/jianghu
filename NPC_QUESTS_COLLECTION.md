# 第一部分：临渊城 NPC 支线

## NQ-LY-01「铁匠的执念」

| 属性 | 详情 |
|------|------|
| **触发** | 与天工坊铁匠大师对话 |
| **地点** | 临渊城商业区·天工坊 |
| **前置** | 无 |
| **奖励** | 专属武器打造权（打折 20%） |

**任务流程：**

```
【触发】
铁匠大师：（叹气）「又一块好铁……可惜了。」

主角：「大师，怎么了？」

铁匠大师：「年轻人，你懂铸剑吗？」
主角：「不懂，但我想学。」

【铁匠大师讲述】
- 他年轻时铸过一把绝世好剑
- 但那把剑被一个恶人抢走，杀了很多人
- 从那以后，他再也不铸好剑了
- 只打普通兵器

【选择】
A) "那把剑在哪？我帮你找回来。"（正义线，推荐）
B) "铸剑本身没错，错的是用剑的人。"（哲理线）
C) "不铸好剑，你的手艺不就荒废了？"（务实线）

【选择 A 后续】
铁匠大师：「你能帮我找回来？」
主角：「给我线索，我去。」

【线索】
- 那把剑叫「断魂」
- 被一个名叫"血刀"的杀手持有
- 血刀出没于西北荒漠

【任务：寻找断魂剑】
- 前往西北荒漠
- 追踪血刀的踪迹
- 需要击败血刀（BOSS 战）

【BOSS 战：血刀】
难度：高
武功：《血刀十三式》
特殊能力：「吸血」（每次攻击回复 10% HP）

【胜利】
获得断魂剑。
可以选择：
  A) 还给铁匠大师（好感 +50，解锁隐藏武器打造）
  B) 自己留着（获得神器「断魂」，但铁匠大师再也不理你）

【推荐选择 A】
铁匠大师：（老泪纵横）「这把剑……终于回来了。」
铁匠大师：「年轻人，从今以后，你要什么武器，我免费给你打！」

解锁：专属武器打造（所有武器 20% 折扣）
获得成就：「信义之士」
```

## NQ-LY-02「百晓生的秘密」

| 属性 | 详情 |
|------|------|
| **触发** | 在商业区与百晓生对话 3 次以上 |
| **地点** | 临渊城商业区 |
| **前置** | 悟性 ≥ 50 |
| **奖励** | 隐藏情报库访问权 |

**任务流程：**

```
【触发】
百晓生：「年轻人，你来了三次了。」
百晓生：「你是不是觉得，我卖的情报不够劲爆？」

主角：「……被你看出来了。」

百晓生：（笑）「我卖的那些，都是给普通人看的。」
百晓生：「真正的好东西……不是一般人能看的。」

【选择】
A) "我想看真正的情报。"（好奇）
B) "多少钱？"（务实）
C) "你是不是有什么条件？"（洞察，推荐）

【选择 C 后续】
百晓生：（眼神一亮）「聪明。」
百晓生：「我有一个忙，需要人帮。」

【百晓生的请求】
- 他在追查一个神秘组织
- 这个组织在各大城镇都有眼线
- 他需要一个人去收集线索

【任务：收集 5 个线索】
线索 1：临渊城黑市的异常交易
线索 2：姑苏城园林区的深夜聚会
线索 3：燕山城军营的失踪士兵
线索 4：汴京城地下擂台的神秘冠军
线索 5：蓬莱城的海外密使

【每个线索需要完成一个小任务】
- 临渊城：潜入黑市，偷听交易
- 姑苏城：夜间跟踪园林区的可疑人员
- 燕山城：调查失踪士兵的最后行踪
- 汴京城：挑战地下擂台冠军
- 蓬莱城：与海外密使交涉

【全部完成后】
百晓生：「好！这些线索足够了！」
百晓生：「作为回报，我给你看我的秘密。」

【百晓生的隐藏情报库】
解锁：
  - 各门派掌门秘密
  - 朝廷官员贪腐证据
  - 隐藏地点地图
  - 稀有武功获取途径

获得成就：「情报达人」
```

## NQ-LY-03「说书人的故事」

| 属性 | 详情 |
|------|------|
| **触发** | 在客栈区听说书人讲故事 |
| **地点** | 临渊城客栈区 |
| **前置** | 无 |
| **奖励** | 隐藏武功线索 |

**任务流程：**

```
【触发】
说书人：「各位看官！今天讲一个真事！」
说书人：「三十年前，有个年轻人——」

【说书人讲故事】
- 三十年前，一个年轻人从西域来中原
- 他武功盖世，但没有门派
- 他打败了各派高手，却没有一个人知道他叫什么
- 后来……他消失了

主角：「后来呢？」

说书人：（叹气）「后来……就没人知道了。」
说书人：「但我听说……他留下了一部武功秘籍。」
说书人：「在一个谁都想不到的地方。」

【选择】
A) "秘籍在哪？"（直接）
B) "那个年轻人是谁？"（追问身份）
C) "你说的'谁都想不到的地方'是哪里？"（推理，推荐）

【选择 C 后续】
说书人：「嗯……你有意思。」
说书人：「不告诉你。」
说书人：「但如果你帮我一个忙——」

【说书人的请求】
- 他想收集一个故事
- 关于「江湖上最感人的故事」
- 需要主角去各地收集

【任务：收集 5 个感人故事】
故事 1：苏清商——听雨楼母女的传承
故事 2：药辛夷——药王谷的医者仁心
故事 3：铁山——铁血盟的兄弟情义
故事 4：赵小满——小乞丐的温暖
故事 5：林晚照——黄泉教的悲剧

【每个故事需要完成对应的好感度任务】

【全部完成后】
说书人：「好故事……都是好故事。」
说书人：「作为回报——」

说书人：「那个年轻人留下的秘籍——」
说书人：「在临渊城城墙的第三块砖下面。」

【获得隐藏武功】
📜 《无名剑法》
  - 无门无派的绝世剑法
  - 共 13 式，每式都有不同的意境
  - 领悟条件：悟性 ≥ 70

获得成就：「故事收集者」
```

# 第二部分：姑苏城 NPC 支线

## NQ-GS-01「丝绸商人的难题」

| 属性 | 详情 |
|------|------|
| **触发** | 与姑苏城丝绸商人对话 |
| **地点** | 姑苏城商业区 |
| **前置** | 无 |
| **奖励** | 银两 3000 + 丝绸 ×10 |

**任务流程：**

```
【触发】
丝绸商人：「叶少侠！你来得正好！」
丝绸商人：「我有一批丝绸要运到北方，但路上有山贼！」

【选择】
A) "我帮你护送。"（帮忙）
B) "多少钱？"（讲价）
C) "你可以走水路。"（出主意，推荐）

【选择 C 后续】
丝绸商人：「水路？但水运比陆运贵三倍啊！」
主角：「但安全。」
主角：「而且——我可以帮你找便宜的船家。」

【任务：找到便宜的船家】
- 前往姑苏城码头
- 与三个船家讨价还价
- 需要口才 ≥ 50 才能拿到最低价

【找到船家后】
- 丝绸走水路运输
- 途中遇到水贼（战斗）
- 保护货物安全到达

【战斗：水贼】
难度：中等
敌人：水贼 × 5（水战，身法影响较大）

【胜利】
丝绸商人：「太好了！货物安全到达！」
丝绸商人：「这是你的报酬！」

获得：
  - 银两 3000
  - 丝绸 ×10（可卖 5000 两）
  - 姑苏城商人好感 +20

解锁功能：
  - 丝绸商人的「贸易任务」
  - 可以通过买卖丝绸赚取差价
```

## EQ-01「悬崖下的老人」

| 属性 | 详情 |
|------|------|
| **触发** | 在大地图随机探索（5% 概率） |
| **地点** | 随机（每次开局不同） |
| **前置** | 无 |
| **奖励** | 随机武功 / 属性提升 |

**任务流程：**

```
【触发】
你在山中行走，突然听到悬崖下有声音。

往下看——
一个白发老人坐在悬崖下的平台上，正在钓鱼。

主角：「老人家，你怎么下去的？」
老人：「走过来的。」

主角：「……走下来的？」
老人：「对啊。」

【选择】
A) "请教我怎么下去。"（拜师）
B) "这地方能钓到鱼吗？"（闲聊）
C) "你是谁？"（追问）

【选择 A 后续】
老人：「想学？可以。」
老人：「但你得先回答我一个问。"

【老人的问题】
「你觉得，武功是用来做什么的？」

回答选项：
A) "用来打架。"（直白）
B) "用来保护重要的人。"（守护）
C) "武功本身就是目的。"（武道）
D) "武功……和写代码一样，都是解决问题的工具。"（程序员，推荐）

【选择 D 后续】
老人：（愣住）「……写代码？」
主角：「呃，就是一个比喻。」
老人：（大笑）「有意思！你这个年轻人有意思！」

老人：「我已经很久没遇到有意思的人了。」
老人：「好，我教你一个武功。」

【获得随机武功】
（根据 RNG 决定）
  30%：随机门派基础武功
  20%：稀有武功
  10%：绝世武功
  40%：属性提升（全属性 +5）

老人：「好了。你走吧。」
主角：「您呢？」
老人：「我继续钓鱼。」

【离开后返回——老人消失了】
平台上空空如也，没有老人，没有鱼竿。
仿佛从来没有来过。

获得成就：「悬崖奇遇」
```

## MQ-01「救或不救」

| 属性 | 详情 |
|------|------|
| **触发** | 在大地图行走时随机触发 |
| **地点** | 路边 |
| **前置** | 无 |
| **奖励** | 声望变化（善/恶） |

**任务流程：**

```
【触发】
路边躺着一个受伤的旅人。

旅人：「救……救我……」

【观察】
- 他穿着普通
- 伤口是剑伤
- 周围没有追兵

【选择】
A) 救他（消耗药品，善良线）
B) 不救，继续走（冷漠线）
C) 先搜身再决定（谨慎/贪婪线）

【选择 A — 救他】
你用药品治疗他。

旅人：「谢谢……谢谢……」
旅人：「我是……商人。被山贼抢劫了。」
旅人：「这个给你，算是报答。」

获得：
  - 银两 500
  - 声望 +10
  - 后续可能触发「商人的报答」任务

【选择 C — 先搜身】
你搜了他的身。

发现：
  - 他怀里有一封信
  - 信上写着「黄泉教」

他根本不是商人！是黄泉教徒！

【战斗：黄泉教徒】
难度：低（他已经受伤）

获得：
  - 黄泉教密信
  - 声望 -5（因为你搜身了）
  - 但获得了情报

⚠️ 这个选择告诉你：不是所有求助的人都值得信任。
```

## MQ-02「富人的请求」

| 属性 | 详情 |
|------|------|
| **触发** | 在城镇遇到富人 |
| **地点** | 随机城镇 |
| **前置** | 无 |
| **奖励** | 银两（但影响道德值） |

**任务流程：**

```
【触发】
一个富人在路边拦住你。

富人：「少侠！帮我一个忙！」
富人：「我的一个欠债人跑了，帮我追回来！」

【追问】
主角：「他欠你多少？」
富人：「五百两。」
主角：「他为什么欠你？」
富人：「……借的。」

【选择】
A) 帮他追债（银两 +200，道德 -10）
B) 先了解情况（中立）
C) 拒绝（道德 +5，得罪富人）

【选择 B 后续】
你找到了那个欠债人。

欠债人：「少侠！我不是故意不还的！」
欠债人：「那个富人是放高利贷的！」
欠债人：「我借了一百两，他要我还五百两！」
欠债人：「我全家都快饿死了！」

【最终选择】
A) 帮富人追债（道德 -20，银两 +500）
B) 帮欠债人（道德 +20，得罪富人，被追杀）
C) 替欠债人还一百两（银两 -100，道德 +30，推荐）

【选择 C 后续】
你替欠债人还了一百两。

欠债人：「少侠！你是好人！」
欠债人：「我以后一定报答你！」

获得：
  - 道德 +30
  - 欠债人一家的感激
  - 后续可能触发「报恩」任务

⚠️ 但你的银两减少了 100。
⚠️ 做好事是要付出代价的。
```
