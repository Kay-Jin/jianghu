## 一、动作设计核心理念

### 1.1 武侠动作四阶段

每个武功招式分为四个阶段：

```
【起势】→ 【蓄力】→ 【出招】→ 【收势】

起势（0.15s）：角色进入预备姿态
  - 暗示即将出招
  - 给对手反应窗口

蓄力（0.2-0.5s）：力量聚集
  - 武器/身体发光或产生特效
  - 时间越长，威力越大（可选）

出招（0.1-0.3s）：招式释放
  - 主要伤害/效果发生
  - 特效最密集

收势（0.15-0.3s）：回归待机
  - 角色回到自然姿态
  - 可能有残影/余波效果
```

### 1.2 武器风格差异化

| 武器 | 节奏特点 | 视觉关键词 |
|------|----------|-----------|
| 剑 | 轻盈、灵动、连绵不断 | 水、风、月 |
| 刀 | 刚猛、大开大合、一击必杀 | 火、雷、山 |
| 拳掌 | 近战爆发、连击 | 刚、震、崩 |
| 棍 | 范围控制、扫荡 | 龙、扫、轮 |

## 二、断剑山庄 · 剑法

**整体风格：** 刚中带柔，断剑之意

### 2.1 断剑剑法（基础）

```
【起势】
  右手持剑，剑尖斜指地面
  身体微侧，左腿前弓右腿后蹬

【蓄力】（0.2s）
  剑身微微抬起
  剑尖出现淡淡银白色光晕

【出招】（0.15s）
  从右上向左下斜劈
  一道银白色半月形剑气飞出
  剑气宽度约角色身高的 1/3

【收势】（0.15s）
  剑收回身侧，剑尖朝下
  剑气残影在空气中缓缓消散

【Godot 实现】
  - 起势：旋转角色骨骼到预备姿态
  - 蓄力：剑身 Sprite 添加发光 shader
  - 出招：Line2D 画半月形轨迹，Line2D.width = 30
  - 收势：粒子效果缓缓消散
```

### 2.2 断水式

**灵感：** "抽刀断水水更流"（李白）—— 反向用之，以断剑之意斩断水流

```
【起势】
  双手握剑（或右手持剑，左手捏剑诀）
  剑横于身前，水平位置

【蓄力】（0.3s）
  剑身从中间出现裂痕般的蓝光
  地面出现水波纹扩散效果
  角色周围气流旋转

【出招】（0.2s）
  从中间向两侧同时展开
  两道蓝色剑气分别向左右飞出
  剑气呈"一"字直线，贯穿整行
  剑气经过的地面出现短暂的冰裂纹理

【收势】（0.2s）
  剑收回，裂痕光效沿剑身汇聚到剑尖
  剑尖一闪，光效消失
  地面冰裂纹理缓缓消退

【特效配色】
  主色：冰蓝 (#4A90D9)
  辅助：银白 (#E0E0FF)
  地面效果：淡蓝冰裂
```

### 2.3 断风式

**灵感：** "大风起兮云飞扬"—— 以断剑之意斩断疾风

```
【起势】
  身体前倾，剑向后拉
  衣摆无风自动

【蓄力】（0.25s）
  角色周围气流开始旋转
  形成小型旋风（粒子效果）
  剑身被旋风包裹

【出招】（0.15s）
  快速前冲突刺 + 上挑
  突刺阶段：角色向前位移 2 格
  上挑阶段：剑从下向上挥出，形成弧形剑气
  剑气颜色为青白色

【收势】（0.15s）
  前冲惯性使角色多滑行 0.5 格
  旋风效果消散
  角色回到站立姿态

【特效配色】
  主色：青白 (#A0D8EF)
  旋风：半透明白色粒子
  位移：角色身后出现短暂残影
```

### 2.4 断月式

**灵感：** "举杯邀明月，对影成三人"—— 月影与剑影交织

```
【起势】
  剑举过头顶，剑尖朝天
  身体后仰，形成反弓姿态

【蓄力】（0.4s）
  剑尖出现月牙形光效
  月牙光效逐渐变大
  地面出现月光照射的光斑

【出招】（0.2s）
  剑从头顶向前方弧形挥下
  月牙形剑气以扇形扩散
  覆盖前方锥形区域（约 120 度角）
  剑气颜色为银白色带淡蓝

【收势】（0.25s）
  剑挥到最低点后缓缓抬起
  月牙剑气在空气中旋转一周后消散
  地面月光光斑缓缓消退

【特效配色】
  主色：银白月光 (#D0D0F0)
  月牙：淡蓝 (#80B0E0)
  地面光斑：半透明白色
```

### 2.5 断魂式

**灵感：** "十步杀一人，千里不留行"—— 一击必杀的决绝

```
【起势】
  剑收于身后，全身紧绷
  角色微微下蹲
  周围空气凝固（画面轻微变暗 0.1s）

【蓄力】（0.5s）
  剑身发出暗紫色光芒
  角色周围出现黑色气流
  画面震动一次（暗示力量积累）
  敌方单位出现被锁定的红色标记

【出招】（0.1s — 极快！）
  瞬间突刺到目标面前（瞬移感）
  一剑穿心（剑从角色位置直达目标）
  紫色冲击波从命中点扩散
  目标被击退 1 格 + 眩晕效果

【收势】（0.3s）
  角色出现在目标身后（背身收剑）
  缓缓转身
  目标身上出现紫色剑气残留，缓缓消散

【特效配色】
  主色：暗紫 (#6030A0)
  锁定标记：红色十字
  冲击波：紫色环形扩散
  画面效果：出招瞬间画面闪烁 0.05s
```

### 2.6 断剑十三式（终极）

**灵感：** "剑来"—— 万千剑意汇聚

```
【起势】
  双手持剑，剑尖指天
  全身真气爆发（全身发光 0.3s）
  地面出现巨大剑阵纹路

【蓄力】（0.6s）
  空中出现 13 道剑影（半透明）
  剑影围绕角色旋转
  每道剑影颜色略有不同
  地面剑阵纹路逐渐亮起

【出招】（0.3s）
  13 道剑影同时射向目标
  每道剑影命中时产生一次小爆炸
  最后本尊一剑从天而降
  终极一剑命中时：
    - 画面闪白 0.1s
    - 地面出现巨大剑痕
    - 目标被击退 2 格 + 破甲

【收势】（0.5s）
  角色缓缓落地，单膝跪地（消耗巨大）
  1-2 秒后缓缓站起
  地面剑阵纹路缓缓消退
  空中残留剑影化为光点飘散

【特效配色】
  13 道剑影：13 种不同颜色
  最终一击：纯白色
  剑阵：金色纹路
  地面剑痕：黑色焦痕
```

## 三、听雨楼 · 剑法

**整体风格：** 轻灵飘逸，以柔克刚

### 3.1 听雨剑法（基础）

```
【起势】
  剑尖斜指前方，身体轻盈侧立
  如雨中舞剑的姿态

【蓄力】（0.2s）
  周围出现雨滴特效（小蓝点下落）
  剑身出现水波纹

【出招】（0.15s）
  快速连续三刺
  每次刺击产生小水花
  三刺之间间隔 0.05s（连绵感）

【收势】（0.1s）
  剑尖点地，雨滴效果消散

【特效配色】
  主色：水蓝 (#50A0D0)
  雨滴：半透明浅蓝
  水花：白色小粒子
```

### 3.2 细雨绵绵

```
【起势】
  剑在身前画圆
  身体随之旋转半圈

【蓄力】（0.25s）
  周围雨滴变得密集
  剑身周围出现水幕

【出招】（0.3s）
  连续 3-5 次斩击
  每次斩击方向不同（上→斜→下→平）
  形成绵密的攻击网
  每次斩击都有连击判定

【收势】（0.15s）
  最后一剑斜指天空
  雨幕缓缓消散

【特效配色】
  雨滴密度：比普通高 3 倍
  斩击轨迹：蓝色弧线残影
```

### 3.3 骤雨狂风

```
【起势】
  剑收于腰间
  身体大幅后仰

【蓄力】（0.35s）
  周围出现狂风效果（水平粒子流）
  雨滴被风吹成水平方向
  角色衣摆剧烈飘动

【出招】（0.2s）
  向前方释放直线剑气
  剑气宽度较窄但穿透力极强
  剑气经过的路径上出现密集雨滴
  命中后附加沉默效果（敌人无法行动 1 回合）

【收势】（0.2s）
  狂风效果逐渐减弱
  雨滴恢复正常下落
  角色回到自然站姿

【特效配色】
  风：半透明白色水平粒子
  剑气：深蓝 (#3060A0)
  沉默效果：灰色锁链缠绕敌人
```

## 四、青囊谷 · 医毒双修

**整体风格：** 绿色治愈 / 紫色剧毒

### 4.1 青囊剑法（基础）

```
【起势】
  剑身缠绕绿色草药藤蔓（装饰效果）
  轻盈的起手姿态

【蓄力】（0.2s）
  剑身绿光增强
  地面出现小草生长的动画

【出招】（0.15s）
  剑身挥舞，绿色剑气飞出
  剑气命中后产生治疗效果（对友方）或毒伤害（对敌方）

【收势】（0.1s）
  藤蔓效果缓缓消退
```

### 4.2 回春掌（治疗）

```
【起势】
  双手合于胸前
  掌心相对

【蓄力】（0.3s）
  掌心之间出现绿色光球
  光球逐渐变大
  周围飘散花瓣（白色/粉色）

【出招】（0.2s）
  双手向前推出
  绿色光波扩散到目标身上
  目标被温暖的绿色光芒笼罩
  HP 数字以绿色 "+XX" 弹出

【收势】（0.2s）
  光波消散
  花瓣缓缓飘落
  目标身上残留绿色光晕

【特效配色】
  主色：翠绿 (#40C040)
  光球：嫩绿 (#80E080)
  花瓣：白粉色
```

### 4.3 七绝针（毒）

```
【起势】
  右手从袖中取出银针
  指尖夹针，对准目标

【蓄力】（0.25s）
  针尖冒出紫色毒气
  目标身上出现紫色标记

【出招】（0.1s — 极快）
  银针飞出（小投射物）
  针命中目标时爆开紫色毒雾
  中毒效果：3 回合持续伤害

【收势】（0.15s）
  毒雾在目标周围缓缓扩散
  角色回到自然站姿

【特效配色】
  毒气：深紫 (#402060)
  毒雾：半透明紫色粒子
  中毒标记：紫色骷髅图标
```

## 五、铁血盟 · 刀法

**整体风格：** 刚猛无俦，力劈华山

### 5.1 铁血刀法（基础）

```
【起势】
  双手握刀（或单手持刀）
  刀扛在肩上
  身体下沉，马步扎实

【蓄力】（0.25s）
  刀身泛起橙红色光芒
  地面出现裂纹（力量感）

【出招】（0.15s）
  从上到下大力劈砍
  刀光呈橙红色弧形
  命中后地面出现裂痕

【收势】（0.2s）
  刀插入地面后拔出
  地面裂痕保留 2 秒

【特效配色】
  主色：橙红 (#D06020)
  地面裂痕：黑色
```

### 5.2 开山劈

```
【起势】
  刀举过头顶
  全身肌肉紧绷
  角色明显蓄力（身体微微发抖）

【蓄力】（0.4s）
  刀身燃起火焰
  角色周围地面震动（小幅度震动）
  空气扭曲效果（热浪）

【出招】（0.2s）
  全力劈下
  火焰刀光覆盖大范围
  命中后：
    - 目标被击飞 0.5 格
    - 地面出现巨大裂痕
    - 火焰爆炸效果
    - 破甲 20%

【收势】（0.3s）
  刀插在地面，角色喘息
  火焰缓缓熄灭
  1-2 秒后拔出刀

【特效配色】
  火焰：橙红 + 黄色
  热浪：半透明波纹
  爆炸：橙红冲击波
```

## 六、丐帮 · 棍法

**整体风格：** 大巧若拙，以拙胜巧

### 6.1 打狗棒法（基础）

```
【起势】
  棍拄地
  身体微佝（丐帮风格）
  看似随意，实则暗藏杀机

【蓄力】（0.2s）
  棍身出现淡黄色真气
  棍尖微微颤动

【出招】（0.15s）
  快速横扫
  棍风呈淡黄色弧线
  命中后目标减速（被击中后踉跄）

【收势】（0.15s）
  棍收回身前
  恢复到拄棍姿态

【特效配色】
  真气：淡黄 (#D0C060)
  减速效果：黄色锁链
```

### 6.2 亢龙有悔

**灵感：** 降龙十八掌之一—— "亢龙有悔，盈不可久"

```
【起势】
  棍横于胸前
  身体后仰蓄力
  隐约可见龙形气劲

【蓄力】（0.4s）
  金色龙形气劲从棍身涌出
  龙形气劲围绕角色盘旋
  地面出现龙爪印记

【出招】（0.25s）
  棍向前猛击
  金色龙形气劲冲出
  龙形气劲贯穿直线范围
  命中后击退 1 格

【收势】（0.2s）
  龙形气劲消散
  棍收回
  地面龙爪印记缓缓消退

【特效配色】
  龙形：金色 (#F0C040)
  气劲：金 + 白
  龙爪印：金色纹路
```

## 七、少林 · 拳掌

**整体风格：** 庄严厚重，佛光普照

### 7.1 罗汉拳（基础）

```
【起势】
  双手合十
  微微低头（敬礼）
  随后展开拳架

【蓄力】（0.2s）
  拳头泛起淡金色佛光
  身后出现淡淡佛影

【出招】（0.15s）
  直拳出击
  拳风呈金色
  命中时发出"铛"的声音

【收势】（0.15s）
  收回拳架
  佛光消散

【特效配色】
  佛光：淡金 (#E0D080)
  佛影：半透明金色轮廓
```

### 7.2 金钟罩（防御）

```
【起势】
  双手合十于胸前
  闭眼

【蓄力】（0.3s）
  身体周围出现金色钟形光罩
  光罩上有经文流转（文字特效）

【出招】（0s — 防御技能，无攻击动作）
  光罩持续 1 回合
  免疫所有伤害
  光罩被击中时发出金属碰撞声

【收势】（0.2s）
  光罩碎裂为金色光点
  角色睁眼

【特效配色】
  金钟：半透明金色
  经文：金色文字环绕
  碰撞：金色火花
```

## 八、华山 · 剑法

**整体风格：** 潇洒飘逸，剑意纵横

### 8.1 华山剑法（基础）

```
【起势】
  剑指前方
  身姿挺拔，气度不凡

【蓄力】（0.2s）
  剑身泛起紫色剑气
  角色周围出现紫色气流

【出招】（0.15s）
  快速连刺 2-3 次
  每次刺击产生紫色剑气
  暴击时剑气变为金色

【收势】（0.1s）
  剑收回
  剑气消散
```

### 8.2 紫霞神功

```
【起势】
  剑收于腰间
  双手结印
  深吸一口气

【蓄力】（0.4s）
  角色全身被紫色霞光笼罩
  霞光逐渐变浓
  角色属性提升（攻击力 +50% 显示在头顶）

【出招】（0s — 增益技能）
  紫色霞光持续 3 回合
  角色周围有紫色光环旋转

【收势】（0.2s）
  收印
  霞光缓缓消退

【特效配色】
  紫霞：淡紫 (#C080E0)
  光环：紫色旋转
  增益标记：紫色箭头↑
```

## 九、黄泉教 · 毒术

**整体风格：** 阴森恐怖，死亡气息

### 9.1 黄泉剑法（基础）

```
【起势】
  剑斜指地面
  剑身缠绕黑色雾气

【蓄力】（0.2s）
  黑雾变浓
  剑身出现紫色毒纹
```

## 十二、Godot 4.3 动画状态机设计

### 12.1 基础状态机结构

```gdscript
# martial_art_state_machine.gd
extends Node

enum State {
    IDLE,        # 待机
    PREPARE,     # 起势
    CHARGE,      # 蓄力
    ATTACK,      # 出招
    RECOVER,     # 收势
    HIT,         # 受击
}

var current_state: State = State.IDLE
var state_timer: float = 0.0
var current_martial_art: Dictionary = {}

# 每个状态的持续时间（秒）
var state_durations = {
    State.PREPARE: 0.15,
    State.CHARGE: 0.3,
    State.ATTACK: 0.15,
    State.RECOVER: 0.2,
}

func _process(delta):
    state_timer += delta
    
    match current_state:
        State.PREPARE:
            _update_prepare(state_timer)
        State.CHARGE:
            _update_charge(state_timer)
        State.ATTACK:
            _update_attack(state_timer)
        State.RECOVER:
            _update_recover(state_timer)
    
    # 检查是否进入下一阶段
    if state_timer >= state_durations.get(current_state, 0):
        _next_state()
        state_timer = 0.0

func start_martial_art(art_data: Dictionary):
    """开始施展武功"""
    current_martial_art = art_data
    current_state = State.PREPARE
    state_timer = 0.0
    
    # 播放起势动画
    _play_prepare_animation(art_data)

func _next_state():
    match current_state:
        State.PREPARE:
            current_state = State.CHARGE
            _play_charge_animation(current_martial_art)
        State.CHARGE:
            current_state = State.ATTACK
            _play_attack_animation(current_martial_art)
        State.ATTACK:
            current_state = State.RECOVER
            _play_recover_animation(current_martial_art)
        State.RECOVER:
            current_state = State.IDLE
            _play_idle_animation()

func _update_prepare(timer: float):
    """起势阶段更新"""
    var progress = timer / state_durations[State.PREPARE]
    # 插值到起势姿态
    _blend_to_pose("prepare_pose", progress)

func _update_charge(timer: float):
    """蓄力阶段更新"""
    var progress = timer / state_durations[State.CHARGE]
    # 特效强度随进度增加
    _update_charge_effects(progress)

func _update_attack(timer: float):
    """出招阶段更新"""
    var progress = timer / state_durations[State.ATTACK]
    # 播放攻击动画
    _play_attack_motion(current_martial_art, progress)
    # 计算伤害（只在出招中段判定）
    if progress > 0.3 and progress < 0.7:
        _deal_damage()

func _update_recover(timer: float):
    """收势阶段更新"""
    var progress = timer / state_durations[State.RECOVER]
    # 插值回到待机姿态
    _blend_to_pose("idle_pose", progress)
    # 特效消散
    _fade_effects(1.0 - progress)
```
