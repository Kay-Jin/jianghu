## 场景 1：义父密室

**画面：** 断剑山庄地下密室，墙上挂着旧画卷和一把残破的长剑。

**【义父】**
> 「你既已成年，有件事该告诉你了。」
> 「你并非我的亲生儿子。」

**【主角独白】**
> 「我知道我不是你亲生的……但我没想到你会直接说出来。」

**义父：** 「三个月前你在山中失踪，那不是意外。」
义父： 「有人要杀你。我找到你时，你已经……」
义父： 「你已经不是原来的你了。」

**【主角独白】**
> 「他知道？他知道我是穿越者？」

**【选择】**
```
A) "你知道我不是原来那个人？"（坦诚）
B) "有人要杀我？是谁？"（追问危险）
C) "你到底是谁？"（质疑义父身份）
```

**【选择 A 后续】**

义父： 「你眼神变了。以前你看我，是看父亲的眼神。」
义父： 「现在你看我，是看陌生人的眼神。」
义父： 「但我不在乎你是谁。你是我义子，这就够了。」

**【主角独白】**
> 「这个 NPC……比我见过的任何一个游戏角色都有深度。」

**义父：** 「你的真名，叫叶长风。」
义父： 「你是前武林盟主叶天行的独子。」

**【画面闪回】**
```
三年前，武林盟主府。
火光冲天，黑衣人涌入。
叶天行（主角生父）：「快走！带长风走！」
一个年轻人抱着婴儿冲出后门——那是年轻时的义父。
```

**义父：** 「三年前，黄泉教夜袭盟主府。」
义父： 「你父亲战死。我带着你杀出重围。」
义父： 「为了不连累你，我改姓叶，将你收为义子，隐居断剑山庄。」

**【主角独白】**
> 「所以……我不是穿越到一个无名小卒身上。」
> 「我穿越到了一个被灭门的大少爷身上。」
> 「这剧情也太狗血了吧……」

## 场景 2：家传秘籍

**画面：** 义父打开密室暗格，取出一本泛黄的秘籍和一个木盒。

**义父：** 「这是你叶家祖传的《山河社稷功》。」
义父： 「你父亲临终前说，叶家武功不能断绝。」
义父： 「但你根基不稳，不可急于修炼。先从基础开始。」

**【获得物品】**
```
📜 《山河社稷功》残卷（共 9 层，当前仅得前 3 层）
📦 叶家信物：断玉佩（一对，你有半块）
```

**【系统提示 — 武功领悟教学】**
```
你获得了家传绝学《山河社稷功》！

当前状态：未领悟（0%）
领悟条件：
  - 悟性 ≥ 50（你当前：XX）
  - 需要基础内力 ≥ 30（你当前：XX）
  - 需要修炼时间：每层 7 天

⚠️ 警告：强行领悟高阶武功可能导致走火入魔！
```

**义父：** 「这半块玉佩，是你父亲留给你的。」
义父： 「另外半块……可能在一个人手里。」
义父： 「如果你想知道全部真相，去找她。」
义父： 「她叫……林素心。」

**【新线索解锁：林素心】**

## 场景 3：寻找林素心

**画面：** 大地图 → 蜀中 → 青城山镇 → 青城后山·隐居小屋

**【主角独白】**
> 「义父只说她住在青城后山，没说具体位置。」
> 「这找人的难度比找 Bug 还高……」

**【探索 — 青城后山】**
```
你在山间寻找林素心的踪迹……

线索 1：路边有新鲜的药篓印记
线索 2：远处有炊烟
线索 3：一只白鹤从山顶飞过

选择追踪方向：
A) 追踪药篓印记（药辛夷可能认识）
B) 往炊烟方向走（有人居住）
C) 跟随白鹤（可能是灵兽引路）
```

**【选择 B — 炊烟方向（推荐路线）】**

**画面：** 竹林深处，一座小院，门前种满草药。

**【老婆婆】**
> （坐在门口择菜）「年轻人，找谁？」

**主角：** 「请问，这里有一位叫林素心的前辈吗？」

**老婆婆：** （动作一顿）「……你找她做什么？」

**【选择】**
```
A) "我是叶天行的儿子。"（亮明身份）
B) "受人之托，来送一封信。"（委婉）
C) "这很重要，请让我见她。"（急切）
```

**【选择 A — 亮明身份】**

老婆婆： （上下打量你）「像。眼睛像你父亲。」
老婆婆： 「进来吧。她在等你。」

## 场景 4：林素心的真相

**画面：** 小院屋内，一个白发苍苍的老妇人坐在榻上，神情平静。

**【林素心】**
> 「你来了。比我想象的晚了一些。」

**主角：** 「前辈认识我？」

**林素心：** 「我认识的是你父亲。至于你……」
林素心： 「你不是他。」

**【主角独白】**
> 「又一个知道的？我这穿越是不是太明显了？」

**林素心：** 「三年前，天行死的那晚，我就在场。」
林素心： 「我是你父亲的师妹。也是……」
林素心： 「也是叶家《山河社稷功》的唯一传人。」

**【画面闪回】**
```
三年前，武林盟主府。
林素心（年轻时）：「师兄！快走！」
叶天行：「素心！你带长风先走！」
林素心：「我不走！要死死在一起！」
叶天行：「叶家不能绝后！走！！」
```

**林素心：** 「你父亲把另外半块玉佩交给了我。」
林素心： 「他说，如果你还活着，总有一天会来找我。」

**【获得物品】**
```
📦 叶家信物：断玉佩（另外半块）
  → 两块玉佩合而为一！解锁隐藏成就「叶家血脉」
```

**林素心：** 「但你要知道的，不只是这些。」
林素心： 「你父亲的死，不只是黄泉教干的。」
林素心： 「背后还有人。」

## 场景 5：更大的阴谋

**画面：** 林素心展开一幅旧信纸，上面有模糊的字迹。

**林素心：** 「这是我从现场找到的。」
林素心： 「信是黄泉教徒写的，但落款不是黄泉教。」
林素心： 「是……」

**【信纸内容】**
```
「……盟主府已清理完毕。叶天行已死。」
「……按计划进行，武林大会前完成部署。」
「……蔡相那边已打点妥当，六扇门不会插手。」
「……金国使者已到，等候下一步指令。」

落款：[被血渍遮盖，仅可见一个"夜"字]
```

**【主角独白】**
> 「蔡京……又是他。」
> 「黄泉教 + 金国 + 朝廷内奸，这是一张多大的网？」
> 「夜……司空无迹？还是别的什么人？」

**林素心：** 「我一直不敢告诉你义父。」
林素心： 「他知道黄泉教，但不知道朝廷的事。」
林素心： 「如果蔡京真的参与其中……」
林素心： 「那这件事，远比你想象的复杂。」

**【选择】**
```
A) "我会查清楚。"（坚定）
B) "我需要更多证据。"（谨慎）
C) "这件事不能告诉义父吗？"（犹豫）
```

**【选择 B — 谨慎路线（推荐）】**

**林素心：** 「对。现在证据不足，不能轻举妄动。」
林素心： 「但你已经开始调查了，对吗？」

**【新任务解锁：「夜字之谜」】**
```
调查方向：
1. 查"夜"字落款的人是谁
2. 查蔡京与黄泉教的关系
3. 查金国使者在中原的活动

⚠️ 此任务为长线任务，将在第 6 章与主线汇聚
```

> 「长风：」

**【新任务解锁：「山河社稷功·完整传承」】**

## 场景 7：身份认同危机

**画面：** 深夜，主角独自在山庄屋顶，望着月亮。

**【主角独白】**
> 「我是叶长风吗？」
> 「还是……我只是占用了叶长风身体的现代人？」
> 「如果我是现代人，那叶长风的灵魂去哪了？」
> 「如果我是叶长风，那我对这个世界的感情，是真的吗？」

**【系统提示 — 内心抉择】**
```
你对自己的身份产生了疑问……

这不是 Bug，这是 Feature。
每个穿越者都会经历这个阶段。

请选择你对身份的认知：
A) "我就是叶长风。过去的不重要。"（认同古代身份）
B) "我是现代人，只是借用了这具身体。"（认同现代身份）
C) "我是谁不重要，重要的是我要做什么。"（行动主义）
```

**【选择 A — 认同古代身份】**
```
主角："不管我的灵魂来自哪里，这具身体、这些人、这个世界……"
主角："现在都是我的。那就好好活下去。"

获得特质：「心安理得」（全属性 +5，修炼速度 +10%）
```

**【选择 B — 认同现代身份】**
```
主角："我始终是那个 996 的项目经理。只是换了个办公室。"
主角："但这个世界的 Bug，我修定了。"

获得特质：「旁观者清」（悟性 +15，发现隐藏线索概率 +20%）
```

**【选择 C — 行动主义（推荐）】**
```
主角："我是谁？这个问题等打败了最终 BOSS 再想吧。"
主角："现在，我有更重要的事要做。"

获得特质：「知行合一」（全属性 +10，战斗经验获取 +15%）
```

**画面：** 次日清晨，义父在练武场等你。

**义父：** 「昨晚没睡好吧？」
义父： 「有些事，想不通就别想了。」
义父： 「你父亲生前常说一句话——」
义父： 「'剑在手，路在脚。想那么多，不如先打好基本功。'」

**【教学战斗 — 与义父切磋】**
```
对手：义父（叶庄主）
难度：中等（义父会放水，但不会太明显）

【战斗提示】
- 尝试使用新学的「破军」
- 注意义父的防御招式
- 不要硬拼，义父经验比你丰富

【战斗胜利】
义父："不错。你父亲要是看到你现在的样子，一定很高兴。"

【战斗失败】
义父："基本功还不够。再来。"
（不影响剧情推进，可反复挑战）
```
