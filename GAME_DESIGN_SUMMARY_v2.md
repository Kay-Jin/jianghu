### 1.3 市场机会

**市面武侠游戏的共同缺点：**
1. 战斗系统单调（回合制游戏通病）
2. 后期内容不足（通关后无事可做）
3. 武功特效简单（缺乏成长反馈）
4. 命名体系雷同（大量抄袭金庸古龙）
5. 女主系统模板化（缺乏个性）

**我们的机会：**
1. ✅ **武功特效分级** - 市面没有，成长可见
2. ✅ **天赋书系统** - 借鉴古龙风云录但原创
3. ✅ **属性互相制约** - 市面没有，必须取舍
4. ✅ **多女主吃醋** - 市面没有，更真实
5. ✅ **终局玩法** - 类似宝可梦，通关后继续玩
6. ✅ **原创命名体系** - 全部原创，不抄金庸古龙
7. ✅ **穿越者主角** - 现代程序员，有梗有吐槽

## 二、我们的差异化定位

### 2.1 核心定位

| 维度 | 定位 |
|------|------|
| **游戏类型** | 2D 回合制 RPG |
| **参考作品** | 古龙群侠传/侠客风云传/古龙风云录 |
| **美术风格** | 2D 写实国风（AI 生成） |
| **叙事风格** | 金庸宏大 + 古龙写意 + 黄易玄妙 |
| **情感基调** | 70% 幽默轻松 + 30% 侠义沉重 |
| **目标平台** | Steam（PC） |
| **定价策略** | ¥48-68 买断制 |
| **开发周期** | 6 个月（24 周） |
| **开发成本** | ≤¥10000 |
| **团队规模** | 2 人（Kay + AI 助理） |

### 2.2 核心创新点（vs 市面游戏）

| 创新 | 描述 | 市面现状 | 我们的优势 |
|------|------|----------|------------|
| **穿越者主角** | 现代程序员穿越 | 无 | 有梗有吐槽，代入感强 |
| **武功特效分级** | Lv.0-Lv.5，成长可见 | 无特效或固定特效 | 成长反馈，一眼看出强弱 |
| **原创命名体系** | 全部原创 | 大量抄袭金庸古龙 | 独特性，避免审美疲劳 |
| **多女主吃醋** | 同时攻略触发事件 | 各攻略各的 | 更真实，有趣 |
| **天赋书系统** | 先天 + 后天 + 隐藏 | 古龙风云录有但失衡 | 更平衡，可更换 |
| **属性互相制约** | 不能全都要 | 属性独立 | 必须取舍，策略性 |
| **终局玩法** | 通关后继续玩 | 通关即结束 | 类似宝可梦，长期驱动 |

### 2.3 目标玩家画像

| 维度 | 描述 |
|------|------|
| **年龄** | 18-35 岁 |
| **性别** | 男性为主（70%），女性为辅（30%） |
| **喜好** | 武侠、RPG、剧情驱动、多结局 |
| **消费习惯** | 愿意为优质单机付费（¥48-68） |
| **游戏时长** | 单次 2-4 小时，总时长 50-100 小时 |

## 三、详细世界观设定

### 3.1 时代背景

**朝代：** 架空宋朝（类似北宋末年）

**时间：** 宣和年间（约公元 1120 年）

**历史背景：**
- 北方：金国崛起，辽国衰落
- 中原：宋朝积弱，奸臣当道（蔡京、童贯等）
- 江湖：武林盟式微，群雄并起
- 民间：百姓困苦，义军暗涌

**关键历史事件（游戏时间线）：**
- 1 年前：海上之盟（宋金联合灭辽）
- 现在：金国开始南下侵宋
- 未来（游戏结局时）：靖康之耻前夕

### 3.2 世界观基调

**金庸的宏大：**
- 家国情怀贯穿主线
- 江湖与朝堂交织
- 个人命运与时代洪流

**古龙的写意：**
- 人物塑造重于招式
- 悬念与反转
- 经典台词风格

**黄易的玄妙：**
- 武道哲学（武道通玄）
- 精神境界描写
- 天人合一的武学理念

## 四、核心系统详细设计

### 4.1 属性系统

**核心原则：** 属性互相制约，不能全都要

| 属性 | 影响 | 制约关系 | 初始范围 |
|------|------|----------|----------|
| **根骨** | 生命值、防御 | ↑根骨 → ↓身法 | 5-15 |
| **内力** | 内力上限、恢复 | ↑内力 → ↓气血上限 | 5-15 |
| **悟性** | 领悟速度、暴击 | ↑悟性 → ↓根骨 | 5-15 |
| **身法** | 速度、闪避 | ↑身法 → ↓防御 | 5-15 |
| **力量** | 攻击、负重 | ↑力量 → ↓身法 | 5-15 |
| **外貌** | NPC 初始好感 | ↑外貌 → ↓内力 | 5-15 |

**属性上限：**
- 单项属性上限：100
- 属性总和上限：300（6 项平均 50）
- 突破上限需要特殊条件（天赋书/奇遇）

**加点策略：**
| 流派 | 主属性 | 副属性 | 牺牲属性 | 适合武功 |
|------|--------|--------|----------|----------|
| **力量流** | 力量 + 根骨 | 悟性 | 身法 | 刀法/拳掌 |
| **身法流** | 身法 + 悟性 | 外貌 | 根骨 | 剑法/暗器 |
| **内力流** | 内力 + 悟性 | 身法 | 力量 | 内功/医术 |
| **平衡流** | 平均 | 无 | 无 | 万金油 |

### 4.2 天赋系统

**天赋类型：**

| 类型 | 数量 | 获取方式 | 可否更换 |
|------|------|----------|----------|
| **先天天赋** | 3 选 1 | 创建角色 | ❌ 不可换 |
| **后天天赋** | 20+ 本 | 探索/任务/购买 | ✅ 可更换 |
| **隐藏天赋** | 5-8 本 | 特殊条件 | ✅ 可更换 |

**装备限制：** 同时装备 3 本后天天赋

**天赋示例：**

| 天赋名 | 类型 | 效果 | 副作用 | 获取方式 |
|--------|------|------|--------|----------|
| **代码之眼** | 先天 | 悟性 +20%，领悟 +30% | 根骨 -10% | 创建角色 |
| **九次轮回** | 先天 | 死亡复活 1 次 | 全属性 -5% | 创建角色 |
| **医者仁心** | 后天 | 治疗效果 +30% | 攻击 -10% | 青囊谷任务 |
| **铁血丹心** | 后天 | 受伤攻击 +20% | 防御 -10% | 铁血盟任务 |
| **浮生若梦** | 隐藏 | 免疫一次致命 | 随机遗忘武功 | 穿越者遗迹 |
| **剑心通明** | 后天 | 剑法威力 +25% | 刀法威力 -50% | 剑法领悟 |
| **左右互搏** | 隐藏 | 可同时使用 2 门武功 | 内力消耗 +50% | 特殊条件 |

### 4.3 武功系统

**武器分类（6 类）：**

| 类型 | 特点 | 适合属性 | 代表门派 |
|------|------|----------|----------|
| **剑** | 平衡，攻守兼备 | 悟性/身法 | 听雨楼/断剑山庄/忘机渡 |
| **刀** | 高伤害，速度慢 | 力量/根骨 | 铁血盟/黄泉教 |
| **长柄** | 范围攻击，距离优势 | 力量/身法 | 丐帮（打狗棒） |
| **短柄** | 高暴击，高闪避 | 身法/悟性 | 唐门（匕首） |
| **暗器** | 远程，可中毒 | 悟性/身法 | 唐门/听雨楼 |
| **拳掌** | 无武器，灵活 | 根骨/力量 | 少林/青囊谷 |

**武学结构：**
```
一门武学 = 多个招式（3-7 个）+ 被动技能（0-2 个）
```

**武功特效分级（Lv.0-Lv.5）：**

| 等级 | 领悟阶段 | 视觉效果 | 音效 |
|------|----------|----------|------|
| Lv.0 | 未入门 | 无特效 | 基础音效 |
| Lv.1 | 初学乍练 | 微弱光晕（30% 强度） | 基础破空声 |
| Lv.2 | 驾轻就熟 | 明显光晕 + 粒子拖尾 | 清晰破空声 |
| Lv.3 | 融会贯通 | 强烈光晕 + 残影 + 空气波动 | 破空 + 回音 |
| Lv.4 | 出神入化 | 华丽光晕 + 环境互动 + 地面痕迹 | 破空 + 回音 + 环境音 |
| Lv.5 | 登峰造极 | 极致光晕 + 粒子实体化 + 专属 BGM+ 全屏震动 | 专属 BGM+ 震动 |

## 六、技术实现方案

### 6.1 游戏引擎

**推荐：Godot**

| 优势 | 说明 |
|------|------|
| 免费开源 | 无授权费用 |
| 2D 友好 | 专为 2D 设计，比 Unity 轻量 |
| GDScript | 类似 Python，易学 |
| 体积小 | 引擎 + 编辑器约 100MB |
| 导出方便 | 支持 Windows/Mac/Linux |

**备选：Unity**（教程多，但体积大）

### 6.2 AI 工具

| 用途 | 工具 | 费用 |
|------|------|------|
| **美术生成** | Midjourney / Stable Diffusion | $10-30/月 / 免费 |
| **音乐生成** | Suno / Udio | $10-30/月 |
| **音效生成** | AI 音效库 | 部分免费 |
| **图片编辑** | GIMP | 免费 |
| **音频编辑** | Audacity | 免费 |

### 6.3 工作流程

```
第 1 步：设计（AI 完成）
├── 策划文档
├── Godot 代码框架
├── AI 提示词
└── 操作指南

第 2 步：生成素材（AI 工具）
├── 美术 → Midjourney/SD
├── 音乐 → Suno/Udio
└── 音效 → AI 音效库

第 3 步：整合（人工）
├── 导入 Godot
├── 调试
└── 优化
```
