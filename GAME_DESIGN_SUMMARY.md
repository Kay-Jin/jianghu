## 一、项目概述

### 1.1 核心定位

| 维度 | 定位 |
|------|------|
| **游戏类型** | 2D 回合制 RPG |
| **参考作品** | 古龙群侠传/侠客风云传/古龙风云录 |
| **美术风格** | 2D 写实国风（AI 生成） |
| **叙事风格** | 金庸宏大 + 古龙写意 + 黄易玄妙 |
| **情感基调** | 70% 幽默轻松 + 30% 侠义沉重 |
| **目标平台** | Steam（PC） |
| **定价策略** | ¥48-68 买断制 |
| **开发周期** | 6 个月（24 周） |
| **开发成本** | ≤¥10000 |
| **团队规模** | 2 人（Kay + AI 助理） |

### 1.2 核心创新点

| 创新 | 描述 | 实现难度 |
|------|------|----------|
| **穿越者主角** | 现代程序员穿越，有梗有吐槽 | 低 |
| **武功特效分级** | Lv.0-Lv.5，成长可见 | 中 |
| **原创命名体系** | 全部原创，不抄金庸古龙 | 低 |
| **多女主吃醋** | 同时攻略会触发吃醋事件 | 中 |
| **天赋书系统** | 先天 + 后天 + 隐藏，可更换 | 中 |
| **属性互相制约** | 属性互相制约，不能全都要 | 低 |
| **终局玩法** | 通关后继续玩，类似宝可梦 | 高 |

### 1.3 故事背景

**时代：** 架空宋朝末年（宣和年间，约 1120 年）

**背景：**
- 北方：金国崛起，即将南下
- 中原：宋朝积弱，奸臣当道
- 江湖：武林盟式微，群雄并起

**主角：** 现代程序员，加班猝死后穿越到古代

**主线：** 三条路线可选（北线家国/东线江湖/西线个人）

**结局：** 8 种多结局（隐退/殉国/继续/返回现代/破碎虚空/一代宗师/富甲一方/孤家寡人）

## 二、核心系统设计

### 2.1 属性系统

**核心原则：** 属性互相制约，不能全都要

| 属性 | 影响 | 制约关系 |
|------|------|----------|
| **根骨** | 生命值、防御 | ↑根骨 → ↓身法 |
| **内力** | 内力上限、恢复 | ↑内力 → ↓气血上限 |
| **悟性** | 领悟速度、暴击 | ↑悟性 → ↓根骨 |
| **身法** | 速度、闪避 | ↑身法 → ↓防御 |
| **力量** | 攻击、负重 | ↑力量 → ↓身法 |
| **外貌** | NPC 初始好感 | ↑外貌 → ↓内力 |

**属性上限：**
- 单项属性上限：100
- 属性总和上限：300（6 项平均 50）
- 突破上限需要特殊条件

### 2.2 天赋系统

**天赋类型：**

| 类型 | 数量 | 获取方式 | 可否更换 |
|------|------|----------|----------|
| **先天天赋** | 3 选 1 | 创建角色 | ❌ 不可换 |
| **后天天赋** | 20+ 本 | 探索/任务/购买 | ✅ 可更换 |
| **隐藏天赋** | 5-8 本 | 特殊条件 | ✅ 可更换 |

**装备限制：** 同时装备 3 本后天天赋

**天赋示例：**

| 天赋名 | 类型 | 效果 | 副作用 |
|--------|------|------|--------|
| **代码之眼** | 先天 | 悟性 +20%，领悟 +30% | 根骨 -10% |
| **九次轮回** | 先天 | 死亡复活 1 次 | 全属性 -5% |
| **医者仁心** | 后天 | 治疗效果 +30% | 攻击 -10% |
| **铁血丹心** | 后天 | 受伤攻击 +20% | 防御 -10% |
| **浮生若梦** | 隐藏 | 免疫一次致命 | 随机遗忘武功 |

### 2.3 武功系统

**武器分类（6 类）：**

| 类型 | 特点 | 适合属性 |
|------|------|----------|
| **剑** | 平衡，攻守兼备 | 悟性/身法 |
| **刀** | 高伤害，速度慢 | 力量/根骨 |
| **长柄** | 范围攻击，距离优势 | 力量/身法 |
| **短柄** | 高暴击，高闪避 | 身法/悟性 |
| **暗器** | 远程，可中毒 | 悟性/身法 |
| **拳掌** | 无武器，灵活 | 根骨/力量 |

**武学结构：**
```
一门武学 = 多个招式（3-7 个）+ 被动技能（0-2 个）
```

**招式类型（6 种）：**
- 伤害型、BUFF 型、DEBUFF 型、多段型、防御型、特殊型

**武功特效分级（Lv.0-Lv.5）：**

| 等级 | 领悟阶段 | 视觉效果 |
|------|----------|----------|
| Lv.0 | 未入门 | 无特效 |
| Lv.1 | 初学乍练 | 微弱光晕（30% 强度） |
| Lv.2 | 驾轻就熟 | 明显光晕 + 粒子拖尾 |
| Lv.3 | 融会贯通 | 强烈光晕 + 残影 + 空气波动 |
| Lv.4 | 出神入化 | 华丽光晕 + 环境互动 + 地面痕迹 |
| Lv.5 | 登峰造极 | 极致光晕 + 粒子实体化 + 专属 BGM+ 全屏震动 |

**属性颜色体系：**

| 属性 | 颜色 |
|------|------|
| 阳刚 | 红色→橙红→金红 |
| 阴柔 | 淡蓝→深蓝→蓝紫 |
| 玄妙 | 白色→银色→白金色 |
| 毒术 | 淡绿→翠绿→墨绿 |
| 医术 | 淡青→青色→青金色 |
| 魔功 | 淡紫→深紫→紫黑 |
| 正道 | 白色→金色→金红 |

**可攻略女主：8 位**

| # | 名字 | 身份 | 入队 | 攻略难度 |

| 3 | 唐雨柔 | 唐门传人 | ✅ | ⭐⭐⭐⭐ |
| 4 | 慕容霜华 | 北国公主 | ✅ | ⭐⭐⭐⭐⭐ |
| 5 | 柳轻烟 | 青楼花魁 | ❌ | ⭐⭐⭐ |

**好感度系统：**

- 吃醋系统：多女主同时攻略触发
- 专属结局：每个女主独立结局
- 后宫结局：隐藏，攻略 5 个以上

### 2.5 开放世界

**三线结构：**
```
序章 → 自由探索 → 三线任选（北/东/西）→ 汇聚 → 终章
```

**三条主线：**
| 路线 | 主题 | 关键 NPC |
|------|------|----------|
| 北线（家国） | 抗金救国 | 铁血盟、朝廷 |
| 东线（江湖） | 门派恩怨 | 各门派、武林盟 |
| 西线（个人） | 身世之谜 | 义父、神秘人 |

**探索驱动：**
- 传闻系统（NPC 透露线索）
- 奇遇系统（随机触发）
- 收集系统（宝可梦式）

### 2.6 终局玩法

**通关后内容：**
1. 继续探索模式（保留所有进度）
2. 二周目继承（部分要素）
3. 周期性活动（武林大会/悬赏/节日）
4. 家园系统（玩家基地）
5. 收集系统（宝可梦式驱动）
6. 挑战系统（试炼塔 100 层）
7. 动态世界（NPC 游历/季节变化）

**目标游戏时长：**
- 主线通关：20-30 小时
- 全收集：50-60 小时
- 多周目：80-100 小时

## 四、技术实现方案

### 4.1 游戏引擎

**推荐：Godot**

| 优势 | 说明 |
|------|------|
| 免费开源 | 无授权费用 |
| 2D 友好 | 专为 2D 设计，比 Unity 轻量 |
| GDScript | 类似 Python，易学 |
| 体积小 | 引擎 + 编辑器约 100MB |
| 导出方便 | 支持 Windows/Mac/Linux |

**备选：Unity**（教程多，但体积大）

### 4.2 AI 工具

| 用途 | 工具 | 费用 |
|------|------|------|
| **美术生成** | Midjourney / Stable Diffusion | $10-30/月 / 免费 |
| **音乐生成** | Suno / Udio | $10-30/月 |
| **音效生成** | AI 音效库 | 部分免费 |
| **图片编辑** | GIMP | 免费 |
| **音频编辑** | Audacity | 免费 |

### 4.3 工作流程

```
第 1 步：设计（AI 完成）
├── 策划文档
├── Godot 代码框架
├── AI 提示词
└── 操作指南

第 2 步：生成素材（AI 工具）
├── 美术 → Midjourney/SD
├── 音乐 → Suno/Udio
└── 音效 → AI 音效库

第 3 步：整合（人工）
├── 导入 Godot
├── 调试
└── 优化
```
