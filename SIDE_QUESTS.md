# 第一部分：支线任务设计原则

## 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **NPC 绑定** | 每个支线必须有具体 NPC，不能是匿名任务板 |
| **队友关联** | 关键支线影响队友好感度/入队/专属剧情 |
| **选择有意义** | 不同选择导致不同结果，影响后续剧情 |
| **奖励不重复** | 不只是银两/经验，还有武功/天赋/好感/隐藏要素 |
| **一次性** | 奇遇/传闻只出现一次，不可反复刷 |
| **蝴蝶效应** | 小选择可能影响大剧情（如救一个人→后期他帮你） |

## 1.2 支线分类

| 类型 | 数量 | 说明 |
|------|------|------|
| **队友专属支线** | 10+ 个（每个队友 1-2 个） | 解锁好感/入队/专属剧情 |
| **NPC 个人支线** | 30+ 个（每个城镇 4-6 个） | 了解 NPC 故事，获得奖励 |
| **门派支线** | 16+ 个（每个门派 2 个） | 拜师后解锁，获得武功 |
| **奇遇支线** | 20+ 个（随机触发） | 隐藏要素，稀有奖励 |
| **传闻支线** | 15+ 个（NPC 对话触发） | 引导探索，解锁隐藏地点 |
| **道德支线** | 10+ 个（善恶选择） | 影响声望/结局 |
| **节日支线** | 6 个（宋代节日） | 限时活动，特殊奖励 |

**总计：100+ 个支线任务**

# 第二部分：队友专属支线

## 2.1 苏清商（女主 1 号 / 听雨楼少主）

### 支线 1：听雨楼危机

**触发条件：** 苏清商好感度≥40，完成第 2 章

**任务流程：**

```
【触发】
苏清商：「听雨楼出了内鬼，情报网被渗透。我需要你的帮助。」

【选择】
A) "义不容辞。"（好感 +10）
B) "有什么好处？"（苏清商冷笑，好感 -5）
C) "内鬼是谁？有怀疑对象吗？"（好感 +5，展现能力）

【调查阶段】
- 前往扬州听雨楼总部
- 调查三个嫌疑人（掌柜/账房/护卫）
- 需要悟性≥60 才能发现线索

【线索 1：账房先生的异常开支】
- 每月有固定银两流向北方
- 追问账房 → 他说被胁迫

【线索 2：护卫的伤痕】
- 左手虎口有新伤（剑伤）
- 追问护卫 → 他说练功受伤

【线索 3：掌柜的密室】
- 发现与黄泉教的密信
- 掌柜就是内鬼！

【对峙】
掌柜：「你以为听雨楼就干净吗？你们知道的太多了...」

【战斗 — 掌柜 + 2 个黄泉教杀手】
难度：中高
奖励：听雨楼贡献 +50

【结局】
苏清商：「谢谢你。听雨楼欠你一个人情。」
苏清商好感 +15
解锁：听雨楼情报网络（可购买稀有情报）
```

**奖励：**
| 奖励 | 说明 |
|------|------|
| 苏清商好感 | +15 |
| 听雨楼贡献 | +50（可兑换情报/武功） |
| 解锁功能 | 听雨楼情报网络 |
| 银两 | 2000 |

### 支线 2：苏清商的过去

**触发条件：** 苏清商好感度≥60，完成"听雨楼危机"

**任务流程：**

```
【触发】
深夜，苏清商独自在听雨楼顶楼，望着月亮。

苏清商：「你怎么来了？」
主角：「看你一个人，来看看。」

苏清商：「...我娘也是听雨楼的人。她在我十岁那年死了。」
苏清商：「死前只说了一句话：'别走我的路'。」

【选择】
A) "你娘为什么会死？"（深入了解）
B) "过去的事就让它过去吧。"（安慰）
C) "所以你现在走的是什么路？"（追问）

【选择 A 后续】
苏清商：「我娘是听雨楼前任楼主。她被...被朝廷的人害死。」
苏清商：「蔡京的人。因为听雨楼掌握了他勾结金国的证据。」

【主角独白】
> "蔡京...又是他。这个人的手伸得真长。"

【后续任务】
- 调查蔡京与听雨楼的恩怨
- 找到当年幸存的听雨楼老人
- 获得关键证据：蔡京通敌的密信副本

【最终选择】
A) 把证据交给朝廷正直官员（正义线）
B) 把证据交给苏清商，让她自己决定（信任线）
C) 公开证据，让天下人知道（搞事线）

【选择 B 后续（推荐）】
苏清商：「你...相信我？」
主角：「我相信你有自己的判断。」
苏清商好感 +20
苏清商：「从今以后，我的命就是你的。」

【解锁】
- 苏清商好感度上限提升至 120（突破 100）
- 解锁专属剧情：「雨夜谈心」
- 解锁连携技：「听雨双剑」（苏清商 + 主角）
```

**奖励：**
| 奖励 | 说明 |
|------|------|
| 苏清商好感 | +20（可突破 100 上限） |
| 连携技 | 「听雨双剑」 |
| 专属剧情 | 「雨夜谈心」 |
| 情报 | 蔡京通敌证据副本 |

## 2.2 药辛夷（女主 2 号 / 青囊谷传人）

### 支线 1：七叶灵芝

**触发条件：** 药辛夷好感度≥30

**任务流程：**

```
【触发】
药辛夷：「大哥！你能帮我一个忙吗？」
主角：「你又去采什么奇怪的草了？」
药辛夷：「不是！我师父中了奇毒，需要七叶灵芝做药引！」

【药辛夷解释】
- 师父在岭南十万大山深处采药时中毒
- 七叶灵芝只生长在瘴气最浓的地方
- 药辛夷自己去过三次，都失败了

【选择】
A) "走吧，我陪你去。"（好感 +10）
B) "瘴气那么危险，你去了三次？"（关心，好感 +15）
C) "七叶灵芝...这名字听着就不好找。"（吐槽）

【探险阶段 — 十万大山】
- 进入瘴气区域（每回合扣血）
- 需要药品/解毒丹
- 途中遇到毒虫/野兽

【场景 1：毒蛇谷】
- 大量毒蛇
- 药辛夷：「别怕！蛇怕雄黄酒！」
- 药辛夷使用雄黄酒驱蛇

【场景 2：毒瘴林】
- 瘴气浓度极高
- 药辛夷：「捂住口鼻！跟着我的药粉走！」
- 药辛夷撒药粉开路

【场景 3：灵芝崖】
- 七叶灵芝长在悬崖边上
- 主角：「这位置...你怎么够得着？」
- 药辛夷：「我有绳子！」

【意外 — 守护者出现】
- 一条百年巨蟒守护灵芝
- BOSS 战：百年巨蟒
- 难度：高（需要合理用武功和药品）

【战斗胜利】
- 获得七叶灵芝
- 药辛夷：「太好了！师父有救了！」

【后续】
- 回青囊谷，药辛夷救师父
- 药王（师父）感谢主角
- 解锁青囊谷专属武功
```

**奖励：**
| 奖励 | 说明 |
|------|------|
| 药辛夷好感 | +20 |
| 药王好感 | +30 |
| 武功 | 《青囊剑法》或《回春掌》 |
| 药品 | 百年灵芝 ×3 / 解毒丹 ×10 |

### 支线 2：药辛夷的秘密

**触发条件：** 药辛夷好感度≥60，完成"七叶灵芝"

**任务流程：**

```

药辛夷：「大哥...我有个秘密一直没说。」

B) "报仇的事交给我。"（保护，好感 +20）
C) "你娘是谁杀的？有线索吗？"（追问）

【调查阶段】
- 药辛夷娘当年死于"幽冥散"（黄泉教独门毒药）

- 前往北方黄泉教
- 林晚照：「你说是我杀的？有趣...但我确实不是。」

【复杂的选择】
A) 接受真相（药辛夷放下仇恨）
B) 不接受，要林晚照偿命（仇恨线）
C) 让药辛夷自己决定（尊重线）

【选择 C（推荐）】
药辛夷：「...谢谢你让我自己选。」
药辛夷：「我不杀她。但我永远不会原谅。」
药辛夷好感 +25
药辛夷：「大哥，你是我最信任的人。」

【解锁】
- 药辛夷专属剧情
- 解锁毒术技能（主角可学）
- 林晚照好感度变化（取决于选择）
```

**奖励：**
| 奖励 | 说明 |
|------|------|
| 药辛夷好感 | +25 |
| 毒术 | 主角可学习基础毒术 |
| 林晚照好感 | 变化（取决于选择） |

【选择 C — 开垦】
- 在燕山开垦荒地
- 需要游戏内 30 天

- 奖励：长期粮草 + 民心

| 铁血盟声望 | +50 |
