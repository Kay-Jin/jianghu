## 1.3 章节剧情概要

### 序章：穿越醒来（教学关）
- 主角在现代办公室加班 → 时空异常 → 穿越到古代
- 荒野醒来，自定义姓名
- 遇到断剑山庄赵执事，被带回山庄
- 教学战斗（山贼来袭）
- 义父相认，得知穿越后身份
- **关键选择：** 诚实 / 装失忆 / 警惕

### 第 1 章：初入江湖
- 断剑山庄修炼，学习基础武功
- **初遇苏清商：** 被黑衣人追赶，主角拔剑相助
- **初遇药辛夷：** 后山采药，搞笑互动
- **初遇赵小满：** 山庄门口小乞丐，给食物后入队
- 客栈听说书人讲江湖事
- 大地图探索开放
- **关键选择：** 帮助苏清商（A/B/C）、给赵小满食物（A/B/C）

### 第 2 章：门派风云
- 大师兄叶孤舟 vs 二师兄花无言夺权
- 主角发现密道内奸（勾结黄泉教 + 金国）
- 跟踪 → 获得密信 → BOSS 战
- 揭露叶孤舟为内奸
- **叶孤舟动机：** 认为宋朝积弱，金国能结束乱世（有深度的反派）
- 义父回归，传《断剑十三式》
- 铁山来访入队
- **关键选择：** 站队（A/B/C/D）、对叶孤舟态度

### 第 3 章：江湖恩怨（大地图开放）
- 大地图完全解锁：9 大区域
- 五条主线全面开放
- **北线开启：** 铁血盟抗金救国
- **东线开启：** 武林盟主大选
- **西线开启：** 义父揭露身世线索
- **第 4 线开启：** 穿越者遗迹线索
- **第 5 线开启：** 家园系统
- 玩家自由选择探索顺序

### 第 4 章：身世之谜（西线专属）
- 发现原主是前武林盟主之子
- 义父揭露避祸收养真相
- 主角面临身份认同危机
- 获得家传武功《山河社稷功》
- 探索原主失踪真相
- **关键选择：** 接受身世 / 追寻真相 / 放下过去

### 第 5 章：武林大会（东线专属）
- 各门派汇聚临渊城
- 主角代表断剑山庄参赛
- 发现盟主候选人夜无痕有问题
- 揭露夜无痕勾结金国阴谋
- 武林大会战斗（多轮比武）
- **关键选择：** 公开揭露 / 暗中调查 / 联合盟友

### 第 6 章：阴谋浮现（多线汇聚）
- 夜无痕真实身份：黄泉教护法 + 宋朝卧底
- 朝廷奸臣蔡京推波助澜
- 金国大军南下，形势危急
- 伙伴可能牺牲（取决于前置选择）
- 主角陷入低谷
- **关键选择：** 如何面对低谷 / 哪些伙伴留下 / 是否寻求穿越者帮助

### 第 7 章：生死抉择（北线专属）
- 金国大军兵临城下
- 江湖人士面临选择：战 or 逃
- 主角做出最终决定
- 联合各方势力，组建联军
- 大规模战斗（多场战役）
- **关键选择：** 死战 / 智取 / 撤退 / 求援

### 第 8 章：终极对决
- 与最终 BOSS 完颜宗望的三阶段决战
- 揭示反派动机（统一天下，结束战乱）
- 主角获胜，但付出代价
- 金国退兵 / 宋朝暂保
- 进入终章结局

### 终章：侠之大者（8 种结局）

| 结局 | 条件 | 描述 |
|------|------|------|
| **A) 隐退江湖** | 好感度高，选择归隐 | 与爱人隐居山林，不问世事 |
| **B) 为国捐躯** | 选择死战 | 战死沙场，成为民族英雄 |
| **C) 继续征程** | 选择留下 | 继续守护江湖，开放结局 |
| **D) 返回现代** | 收集 5 个穿越相关物品 | 回到现代，失去一切武功和伙伴 |
| **E) 破碎虚空** | 武道满级 + 隐藏条件 | 达到传说境界，飞升而去 |
| **F) 一代宗师** | 完成所有门派支线 | 开宗立派，传承武学 |
| **G) 富甲一方** | 完成所有经营支线 | 成为首富，江湖传说 |
| **H) 孤家寡人** | 队友全灭/背叛 | 孤独终老，悲剧结局 |
