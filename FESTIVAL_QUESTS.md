## FQ-01「元宵灯谜会」

| 属性 | 详情 |
|------|------|
| **触发时间** | 正月十五（元宵节） |
| **地点** | 临渊城·商业区 |
| **前置** | 悟性 ≥ 40 |
| **奖励** | 银两 + 隐藏武功线索 |

**任务流程：**

```
【触发】
正月十五，临渊城张灯结彩。

【活动：猜灯谜】
商业区挂了 10 盏花灯，每盏上有一个灯谜。

灯谜列表（答对 5 题以上获奖励）：

灯谜 1：「天下第一家」（打一字）
  答案：赵（宋朝皇帝姓赵）
  
灯谜 2：「一刀两断」（打一武功名）
  答案：断剑十三式
  
灯谜 3：「身在曹营心在汉」（打一成语）
  答案：身在曹营心在汉
  
灯谜 4：「千里之行始于足下」（打一字）
  答案：踵
  
灯谜 5：「武松打虎」（打一地名）
  答案：景阳冈

【答对 5 题以上】
活动主持：「少侠好才学！这是奖品！」

获得：
  - 银两 1000
  - 「灯谜达人」称号
  - 隐藏线索：说书人提到的武功秘籍位置

【答对全部 10 题】
获得额外奖励：
  - 银两 3000
  - 稀有道具「智慧珠」（悟性 +5）
```

## FQ-02「端午龙舟赛」

| 属性 | 详情 |
|------|------|
| **触发时间** | 五月初五（端午节） |
| **地点** | 姑苏城·水乡区 |
| **前置** | 身法 ≥ 40 |
| **奖励** | 声望 + 特殊称号 |

**任务流程：**

```
【触发】
五月初五，姑苏城河道上人山人海。

【活动：龙舟赛】
你可以代表断剑山庄参加龙舟赛！

【比赛规则】
- 4 人一组，你 + 3 个队友
- 赛程 1000 米
- 需要节奏配合（QTE 小游戏）

【可选队友】
  - 铁山（力量型，划桨有力但节奏感差）
  - 赵小满（灵巧型，节奏感好但力量弱）
  - 药辛夷（平衡型）
  - 苏清商（力量 + 节奏都不错）

【比赛过程】
第 1 段：起步（QTE）
第 2 段：中途加速（QTE）
第 3 段：最后冲刺（QTE）

【结果】
第 1 名：声望 +50，银两 2000，「龙舟之王」称号
第 2 名：声望 +30，银两 1000
第 3 名：声望 +10，银两 500
```

## FQ-03「中秋赏月大会」

| 属性 | 详情 |
|------|------|
| **触发时间** | 八月十五（中秋节） |
| **地点** | 临渊城·西湖 |
| **前置** | 任意女主好感 ≥ 50 |
| **奖励** | 女主专属剧情 + 特殊物品 |

**任务流程：**

```
【触发】
八月十五，月圆之夜。

【选择赏月伙伴】
A) 苏清商（冷艳御姐风）
B) 药辛夷（活泼可爱风）
C) 赵小满（温馨治愈风）
D) 林晚照（忧郁文艺风）
E) 独自赏月（孤独线）

【不同伙伴的不同剧情】

A) 苏清商线：
  苏清商：「今天的月亮……真美。」
  主角：「不如你美。」
  苏清商：（脸红）「……油嘴滑舌。」
  苏清商好感 +10
  解锁：「月下谈心」专属剧情

B) 药辛夷线：
  药辛夷：「大哥！你看那个！月亮上是不是有兔子？」
  主角：「……那是环形山。」
  药辛夷：「什么是环形山？」
  主角：「呃……这个解释起来很长。」
  药辛夷好感 +10
  解锁：「中秋趣事」专属剧情

C) 赵小满线：
  赵小满：「大哥！我以前从没看过这么大的月亮！」
  赵小满：「以前在街上，只看到房子……」
  主角：「以后每年中秋，我都陪你看月亮。」
  赵小满好感 +15
  解锁：「小满的第一个中秋」专属剧情

D) 林晚照线：
  林晚照：（沉默）
  主角：「在想什么？」
  林晚照：「我父亲以前，也喜欢赏月。」
  林晚照：「……谢谢你陪我。」
  林晚照好感 +10
  解锁：「月下思念」专属剧情

【获得特殊物品】
🥮 月饼 ×1（中秋限定）
  - 食用后 HP 全满
  - 但只有一个，舍不得吃
```

## FQ-04「春节庙会」

| 属性 | 详情 |
|------|------|
| **触发时间** | 正月初一（春节） |
| **地点** | 各大城镇 |
| **前置** | 无 |
| **奖励** | 多种活动奖励 |

**任务流程：**

```
【触发】
正月初一，全城庆祝。

【庙会活动】

1. 舞狮表演
   - 观看获得「节日气氛」buff（全属性 +5%，持续 1 天）
   
2. 套圈游戏（小游戏）
   - 花 100 两玩一次
   - 套中什么拿什么
   - 可能套到稀有物品！
   
3. 写春联
   - 需要悟性 ≥ 50
   - 写得好获得银两 500
   - 写得不好被嘲笑
   
4. 放烟花
   - 和伙伴一起放烟花
   - 好感度 +5（所有在场伙伴）

5. 领红包
   - 义父给你发红包
   - 随机获得银两 100-1000
```

## FQ-06「重阳登高节」

| 属性 | 详情 |
|------|------|
| **触发时间** | 九月初九（重阳节） |
| **地点** | 各大城镇附近的高山 |
| **前置** | 根骨 ≥ 40 |
| **奖励** | 属性提升 + 隐藏地点 |

**任务流程：**

```
【触发】
九月初九，登高望远之日。

【活动：登山比赛】

各城镇附近的高山：
  - 临渊城：泰山
  - 姑苏城：虎丘
  - 燕山城：燕山
  - 蜀中：青城山

【登山过程】
- 途中遇到其他登山者
- 可以比赛谁先到山顶
- 途中有岔路，选择不同路线发现不同物品

【登顶奖励】
到达山顶后，可以选择「望远」：

望远后获得：
  - 视野开阔，心境提升
  - 全属性 +2（永久）
  - 解锁隐藏地点「山顶洞」

【山顶洞】
洞里有一个老人留下的武功秘籍：
📜 《登高诀》
  - 轻功类武功
  - 提升身法和闪避
  - 领悟条件：根骨 ≥ 45
```
