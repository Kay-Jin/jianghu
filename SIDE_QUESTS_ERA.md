# 🏮 江湖打工人 — 时代背景支线任务

**版本：** v1.0  
**创建日期：** 2026-04-16  
**设计理念：** 参考《燕云十六声》，每个支线都扎根于北宋末年的社会现实

---

# 第一部分：设计原则

## 1.1 《燕云十六声》的启示

《燕云十六声》的支线之所以出色，是因为：
- 每个故事都反映**五代十国时期的社会现实**
- 普通人在乱世中的挣扎与选择
- 不是"帮我找 10 个草药"，而是"我丈夫被征兵三年未归"
- 小人物也有大故事

## 1.2 北宋末年社会现实（游戏时间线：宣和年间，约 1120 年）

| 历史背景 | 游戏内体现 | 支线主题 |
|----------|------------|----------|
| **花石纲** | 皇帝沉迷奇石异花，劳民伤财 | 江南百姓被征采花石的苦难 |
| **金国南下** | 北方难民南逃，边关告急 | 难民流离失所，家园被毁 |
| **奸臣当道** | 蔡京、童贯等人贪腐 | 官员欺压百姓，民不聊生 |
| **方腊起义** | 江南民变 | 被逼造反的农民 |
| **重税盘剥** | 百姓负担沉重 | 卖儿卖女的悲惨故事 |
| **武林式微** | 江湖人士也受朝廷压迫 | 侠客无力的时代困境 |
| **海上之盟** | 宋金联合灭辽，埋下隐患 | 边防空虚，百姓不安 |

## 1.3 支线情感基调

| 基调 | 占比 | 说明 |
|------|------|------|
| **悲凉** | 40% | 普通人在时代洪流中的无力 |
| **温暖** | 25% | 困境中的人性光辉 |
| **讽刺** | 15% | 对贪腐/荒唐的讽刺 |
| **希望** | 10% | 小人物的抗争与坚守 |
| **幽默** | 10% | 轻松调剂（主角吐槽） |

### 📜 「花石纲」—— 一块石头的代价

**触发：** 开封府大相国寺市场，一个江南口音的老人坐在角落，面前放着一块普通石头。

```
【触发】
老人：「这块石头...花了我三条人命。」

【老人讲述】
- 他是苏州花农，姓张
- 三年前，官府来村里征花石
- 他家后院有一块太湖石，被看中
- 为了运这块石头到东京，征了村里 30 个壮劳力
- 路上死了三个（累死/病死/被监工打死）
- 石头运到了，皇帝看了一眼说"不够好"，不要了
- 石头被扔在路边，老人捡了回来

老人：「三條人命啊...就换了一句'不够好'。」

【选择】
A) "这石头...我帮你卖了吧。"（实际帮助）
B) "当今圣上怎会如此荒唐？"（愤怒）
C) "你恨皇帝吗？"（追问）
D) "这石头确实不好看。"（主角吐槽，不合时宜）

【选择 A 后续】
- 主角带石头去找铁匠大师傅
- 铁匠：「这太湖石...雕一雕，能做一方好砚台。」
- 雕成砚台后，卖给书院
- 获得银两 800，给老人
- 老人：「够了...三条人命的抚恤，够了。」

【选择 C 后续】
老人沉默了很久。
老人：「恨？我连恨的资格都没有。」
老人：「我只是个花农。我能恨谁？恨石头？恨路？恨我自己命不好？」
老人：「我只想知道...他们什么时候能停下来。」

【主角独白】

【后续影响】
```

# 第四部分：设计要点总结

## 4.1 每个支线必须包含

| 要素 | 说明 | 示例 |
|------|------|------|
| **时代背景** | 反映北宋末年社会现实 | 花石纲/金国南下/重税 |
| **具体人物** | 有名字、有故事的小人物 | 灯笼张/阿织/王二 |
| **道德选择** | 没有绝对正确答案 | 救一个还是救一群 |
| **蝴蝶效应** | 小选择影响后续大事件 | 救盐工女儿 → 多年后她救人 |
| **主角吐槽** | 项目经理视角的幽默 | "这是供应链问题" |
| **情感共鸣** | 让人记住的故事 | "拔剑容易收剑难" |

## 4.2 避免的设计

| ❌ 避免 | ✅ 改为 |
|---------|---------|
| "帮我找 10 个草药" | "我娘病了，需要草药，但采药的地方被官府封了" |
| "杀 5 只狼" | "狼群下山是因为森林被砍了，砍树的是官府" |
| "送信" | "这封信是一个士兵写给家的，他可能回不去了" |
| "收集材料" | "这些材料是一个老人一辈子的心血" |

## 4.3 情感节奏控制

```
重（花石纲）→ 轻（州桥夜市）→ 重（施粥棚）→ 暖（灯笼张）
重（城墙上的信）→ 轻（蕃商的儿子）→ 重（逃兵）→ 暖（织娘）

每 2-3 个沉重支线，穿插 1 个轻松/温暖的支线。
```
