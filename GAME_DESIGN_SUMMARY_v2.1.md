### 8.1 核心定位 ✅

| 项目 | 确认内容 |
|------|---------|
| **类型** | 2D 回合制 RPG |
| **画面风格** | 2D 写实风格（参考古龙风云录） |
| **❌ 不要** | 下一站江湖页游风 |
| **主角身份** | 项目经理（不是程序员） |
| **开发周期** | 兼顾完成度和质量，尽量缩短（担心热情问题） |
| **梗频率** | 一般 |
| **幽默** | 可以适当多加点 |
| **配音** | ❌ 不要 |

| **北线** | 家国 | 家国情怀、朝廷与江湖 |
