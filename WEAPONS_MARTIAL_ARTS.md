## 一、武器分类系统

### 1.1 六大武器类型

| 类型 | 特点 | 适合属性 | 代表门派 |
|------|------|----------|----------|
| **剑** | 平衡，攻守兼备 | 悟性/身法 | 逍遥渡/听雨楼/断剑山庄 |
| **刀** | 高伤害，速度慢 | 力量/根骨 | 铁血盟/黄泉教 |
| **长柄** | 范围攻击，距离优势 | 力量/身法 | 丐帮（打狗棒）/枪法门派 |
| **短柄** | 高暴击，高闪避 | 身法/悟性 | 唐门（匕首）/刺客 |
| **暗器** | 远程，可中毒 | 悟性/身法 | 唐门/听雨楼 |
| **拳掌** | 无武器，灵活 | 根骨/力量 | 少林/古墓派 |

### 1.2 武器属性

| 属性 | 说明 | 影响 |
|------|------|------|
| **攻击力** | 基础伤害 | 直接影响招式伤害 |
| **速度** | 攻击频率 | 影响行动顺序 |
| **暴击率** | 暴击概率 | 影响爆发 |
| **命中** | 命中率 | 影响稳定性 |
| **重量** | 武器重量 | 影响身法发挥 |
| **耐久** | 耐久度 | 需要维修 |

### 1.3 武器品质

| 品质 | 颜色 | 属性加成 | 获取方式 |
|------|------|----------|----------|
| **普通** | 白色 | 无 | 初始/购买 |
| **优秀** | 绿色 | +10% 攻击 | 任务/打造 |

## 二、武学结构设计

### 2.1 武学组成

**一门武学 = 多个招式**

```
武学结构：
├── 武学名称（如《流云七剑》）
├── 武器类型（如剑）
├── 属性（如阴柔）
├── 领悟阶段（0-5 级）
├── 招式列表（3-7 个招式）
│   ├── 招式 1：基础招式（伤害型）
│   ├── 招式 2：进阶招式（伤害 +BUFF）
│   ├── 招式 3：进阶招式（伤害 +DEBUFF）
│   ├── 招式 4：高级招式（多段伤害）
│   ├── 招式 5：高级招式（伤害 + 强力 BUFF）
│   ├── 招式 6：终极招式（高伤害 + 特效）
│   └── 招式 7：奥义（消耗大，威力极强）
└── 被动技能（0-2 个）
    ├── 被动 1：领悟后自动生效
    └── 被动 2：登峰造极后解锁
```

### 2.2 招式类型

| 类型 | 说明 | 例子 |
|------|------|------|
| **伤害型** | 直接造成伤害 | 流云出岫、断水一击 |
| **BUFF 型** | 给自己加增益 | 流云护体、铁血战意 |
| **DEBUFF 型** | 给敌人加减益 | 听雨迷魂、黄泉诅咒 |
| **多段型** | 多次伤害 | 流云连剑、暴雨梨花 |
| **防御型** | 减少伤害/格挡 | 断剑守势、金刚不坏 |
| **特殊型** | 特殊效果（吸血/闪避/反击） | 黄泉吸血、听雨闪避 |

### 2.3 招式详细结构

**每个招式包含：**

| 属性 | 说明 | 示例 |
|------|------|------|
| **名称** | 招式名 | 流云出岫 |
| **类型** | 伤害/BUFF/DEBUFF/多段/防御/特殊 | 伤害型 |
| **消耗** | 内力消耗 | 20 点 |
| **冷却** | 冷却回合 | 0 回合 |
| **基础伤害** | 伤害倍率 | 100% 攻击力 |
| **附加效果** | BUFF/DEBUFF/特殊 | 无 |
| **解锁条件** | 领悟阶段要求 | 初学乍练可用 |
| **特效等级** | Lv.0-Lv.5 | 随武学等级变化 |

## 三、被动技能系统

### 3.1 被动技能分类

| 分类 | 说明 | 例子 |
|------|------|------|
| **属性加成** | 直接增加属性 | 闪避 +10%、攻击 +20% |
| **战斗增益** | 战斗中生效 | 受伤攻击 +10%、首击暴击 |
| **探索增益** | 探索时生效 | 移动速度 +20%、发现率 +30% |
| **社交增益** | NPC 交互生效 | 好感度 +20%、砍价 +10% |
| **特殊效果** | 特殊机制 | 死亡复活、免疫毒素 |

### 3.2 被动技能获取方式

| 方式 | 说明 | 例子 |
|------|------|------|
| **武学领悟** | 领悟武学后自动获得 | 《流云七剑》→流云身法 |
| **天赋书** | 学习天赋书获得 | 《医者仁心》天赋书 |
| **装备** | 穿戴装备获得 | 听雨剑→听雨剑意 |
| **奇遇** | 特殊事件获得 | 高人传功 |
| **成就** | 达成成就获得 | 连胜 10 场→连胜之心 |

### 3.3 被动技能叠加规则

**叠加原则：**
- 同名被动不叠加（取最高）
- 异名被动可叠加
- 有上限（单项属性上限 +100%）

**示例：**
```
被动 1：流云身法（闪避 +10%）
被动 2：踏雪无痕（闪避 +15%）
被动 3：燕子飞（闪避 +20%）

总闪避加成：10%+15%+20% = +45%（可叠加）
```
