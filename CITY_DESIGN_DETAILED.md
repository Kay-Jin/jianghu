# 🏯 大城市设计文档 — 沉浸感驱动

**版本：** v1.0  
**创建日期：** 2026-04-17  
**设计理念：** 参考《燕云十六声》的沉浸感——城市不是背景板，是活的。

---

## 一、大城市核心要素

### 1.1 "活着的城市"标准

| 维度 | 最低要求 | 理想状态 |
|------|----------|----------|
| NPC 数量 | 15+ 个可交互 | 30+ 个，各有生活 |
| 店铺数量 | 6+ 种类型 | 10+ 种，可经营 |
| 支线任务 | 8+ 个 | 15+ 个，蝴蝶效应 |
| 建筑地标 | 5+ 个 | 10+ 个，可进入 |
| 区域划分 | 4+ 个功能区 | 8+ 个，各有特色 |
| 动态事件 | 3+ 个/次探索 | 8+ 个，随时间变化 |
| 氛围音效 | 5+ 种环境音 | 10+ 种，昼夜变化 |

### 1.2 NPC 生活系统

NPC 不是站着不动的木头桩子，他们有**日常作息**：

```
NPC 日常作息表（以赵铁匠为例）：
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ 06:00  │ 09:00  │ 12:00  │ 14:00  │ 18:00  │ 21:00  │
│ 起床   │ 开店   │ 午休   │ 继续   │ 收摊   │ 回家   │
│ 家门口 │ 铁匠铺 │ 酒楼   │ 铁匠铺 │ 回家   │ 家里   │
└────────┴────────┴────────┴────────┴────────┴────────┘

玩家在不同时间找他，会触发不同对话：
  早上："这么早来打铁？"
  中午："老板不在，吃饭去了"
  晚上："收摊了，明天再来吧"
```

#### 特色系统

**1. 漕运系统**
- 临渊城是水路枢纽
- 可以从码头坐船去江南/东海
- 水路比陆路便宜但有海盗风险

- 玩家可以参加科举考试
- 考中后获得朝廷官职

- 被发现会被六扇门追捕

**4. 不夜城**
- 临渊城没有宵禁
- 夜晚商业区和青楼区更热闹
- 黑市只在深夜开放

## 三、NPC 行为系统设计

### 3.1 NPC 日程表

```gdscript
# npc_schedule.gd
class NPCSchedule:
    var npc_name: String
    var schedule: Dictionary = {}  # {hour: {location, action, dialog_id}}
    
    func get_current_state(hour: int) -> Dictionary:
        # 找到当前时间段的状态
        var closest_hour = 0
        var closest_state = schedule.get(0, {})
        
        for h in schedule:
            if h <= hour and h > closest_hour:
                closest_hour = h
                closest_state = schedule[h]
        
        return closest_state

# 示例：赵铁匠的日程
var zhao_schedule = {
    6: {"location": "home", "action": "起床", "dialog": "zhao_morning"},
    9: {"location": "forge", "action": "开店", "dialog": "zhao_open"},
    12: {"location": "tavern", "action": "午休", "dialog": "zhao_lunch"},
    14: {"location": "forge", "action": "继续打铁", "dialog": "zhao_afternoon"},
    18: {"location": "home", "action": "收摊", "dialog": "zhao_close"},
    21: {"location": "home", "action": "休息", "dialog": "zhao_night"},
}
```
